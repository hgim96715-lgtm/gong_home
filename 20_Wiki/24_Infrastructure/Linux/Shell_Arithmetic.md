##  개념 한 줄 요약  

**"리눅스 쉘(Bash)을 계산기처럼 쓰는 방법."**

* 쉘은 기본적으로 모든 것을 **문자열(Text)** 로 취급합니다.
* 따라서 `1+1`을 입력하면 계산 결과인 `2`가 아니라, 글자 그대로 `1+1`이 출력됩니다.
* 숫자로 계산하고 싶다면 **특수한 문법 `(( ))`** 안에 가둬야 합니다.

---
## 핵심 문법: 이중 괄호 `(( ))` 

가장 권장되는 모던(Modern) 방식입니다.
- 기본 문법: `$(( 수식 ))`

```bash
# 기본 문법: $(( 수식 ))
result=$(( 10 + 20 ))
echo $result  # 30
```

### 사칙연산 기호

- `+` : 더하기
- `-` : 빼기
- `*` : 곱하기
- `/` : 나누기 (몫만 나옴)
- `%` : 나머지 (Modular)

---
## 주의사항 (정수 vs 실수) 

### ❌ Bash는 소수점(실수) 계산을 못 합니다!

```bash
echo $(( 10 / 3 ))
# 결과: 3 (3.333... 이 아니라 정수 부분만 나옴)
```

### ✅ 소수점이 필요할 땐 `bc`를 써야 합니다

만약 정교한 계산이 필요하다면 `bc` (Basic Calculator) 라는 도구를 파이프로 연결해서 써야 합니다.

```bash
# scale=2 : 소수점 둘째 자리까지 보여달라는 뜻
echo "scale=2; 10 / 3" | bc
# 결과: 3.33
```
