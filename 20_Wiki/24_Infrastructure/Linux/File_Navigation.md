## 개념 한 줄 요약

**파일 내비게이션**은 터미널이라는 어두운 미로 속에서 내가 어디에 있는지 확인하고, 원하는 방으로 자유롭게 걸어 들어가는 가장 기본적인 동작이야.

---
## 왜 필요한가? (Why)

**문제:**
- GUI(윈도우/맥 탐색기)에서는 폴더가 눈에 보이지만, CLI(터미널)에서는 내가 지금 어디에 있는지, 옆 방에 뭐가 있는지 전혀 보이지 않아.

**해결:**
- `pwd`로 주소를 확인하고, `ls`로 주변을 살피고, `cd`로 순간이동해야 해.
- 이 세 명령어가 손에 익지 않으면 리눅스라는 미로에서 영원히 길을 잃게 될 거야.

---
## Practical Context (현업 시나리오)

데이터 엔지니어는 하루에도 수십 번씩 서버를 옮겨 다녀.
- "로그가 쌓인 `/var/log` 폴더로 가서 에러를 확인해야지."
- "설정 파일이 있는 `/etc` 폴더로 이동해서 DB 주소를 바꿔야겠어."
- "내 작업 결과물이 저장된 `/home/ubuntu/example` 폴더에 뭐가 들어있지?"
이런 상황에서 이 명령어들은 공기와 같은 존재야.

---
## Code Core Points (핵심 명령어)

가장 많이 쓰는 기본 조합 3대장이야.

1. **`pwd`**: **P**rint **W**orking **D**irectory. 현재 내 위치의 **절대 경로**를 보여줘.
2. **`ls`**: **L**i**s**t. 현재 폴더에 들어있는 파일과 폴더 목록을 보여줘.
3. **`cd`**: **C**hange **D**irectory. 지정한 폴더로 **이동**해.

---
## Detailed Analysis (상세 분석)

실습 서버(`ubuntu` 유저)에 접속했다고 가정하고 흐름을 따라가 보자.

```bash
# 1. 내가 지금 정확히 어디 있지?
pwd
# 결과: /home/ubuntu  <-- (중요) 우리는 이제 일반 유저이므로 홈이 여기야!

# 2. 여기에 뭐가 들어있어? (숨겨진 파일까지 자세히)
ls -al
# -a (all): 숨겨진 파일(.bashrc 등 점으로 시작하는 것)도 보여줘.
# -l (long): 파일 권한, 소유자, 크기, 날짜 등 상세 정보를 리스트로 보여줘.

# 3. 루트(/) 폴더로 가볼까? (절대 경로 이동)
cd /
# 주소의 시작이 / 이면 무조건 뿌리부터 찾는 '절대 경로'야.

# 4. 다시 상위 폴더로 가거나 홈으로 가고 싶어? (상대 경로 & 특수 기호)
cd ~     # 내 방(/home/ubuntu)으로 즉시 복귀 (가장 많이 씀!)
cd ..    # 지금 있는 곳의 바로 위(부모) 폴더로 이동
cd -     # 방금 전에 있었던 '직전 폴더'로 돌아가기 (TV 채널 '이전' 버튼과 같음)

cd /tmp  # tmp 폴더 이동 

# 5. (꿀팁) 멀리 있는 파일 확인하기
ls -lh /var/log
# -h (human-readable): 파일 용량을 보기 편하게(KB, MB, GB) 보여줘. 데이터 엔지니어 필수 옵션!

```

---
## Survival Tip: 탭(Tab) 자동완성

**"절대 모든 글자를 타이핑하지 마라."**

- **방법:** `cd Do` 까지만 치고 **[Tab]** 키를 눌러봐.
- **결과:** `cd Documents/` 라고 자동으로 완성돼.
- **효과:**
    1. 타이핑 속도가 3배 빨라짐.
    2. 오타(대소문자 실수)를 100% 방지함.
    3. 파일이 실제로 존재하는지 미리 알 수 있음 (완성이 안 되면 파일이 없는 것!).

## 초보자가 자주 착각하는 포인트

- **"파일 이름이 맞는데 왜 안 들어가져요?"**
    - 리눅스는 **대소문자**를 칼같이 구분해! 
    - `Downloads`와 `downloads`는 완전 남남이야. 탭(Tab)을 쓰면 해결돼.

- **"`cd`만 쳤는데 아무 일도 안 일어나요."**
    - 에러가 아냐. `cd`만 치면 무조건 **내 홈 디렉토리(`~`)** 로 돌아가게 되어 있어. 
    - 길을 잃었을 때 쓰면 좋아.

- **"슬래시(`/`)를 붙여야 하나요?"**
    - 맨 앞의 `/`는 **"뿌리부터 시작(절대경로)"** 한다는 뜻이고,
    - `/`가 없으면 **"지금 내 위치에서 시작(상대경로)"** 한다는 뜻이야.
    - 헷갈리면 `pwd`로 내 위치를 먼저 확인해!
