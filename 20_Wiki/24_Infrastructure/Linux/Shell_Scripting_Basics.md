## 개념 한 줄 요약

"리눅스 명령어들을 파일 하나에 모아서 순서대로 실행시키는 '자동화 대본(Script)'."

- **Bash (Bourne Again SHell):** 리눅스의 표준 통역사(Shell). 가장 강력하고 널리 쓰입니다.
- **쉘 스크립트 (`.sh`):** 단순한 명령어 나열을 넘어, **조건(if)** 과 **반복(for)** 을 통해 똑똑하게 일처리를 하는 파일입니다.

> ⚠️**핵심 규칙:**
>  Bash는 **"공백(띄어쓰기)"** 이 문법의 생명입니다. (붙여야 할 때와 띄어야 할 때를 엄격히 구분합니다!)

## 핵심 문법 ①: 기본 구조와 실행

### 스크립트의 시작 (Shebang)

모든 스크립트의 첫 줄은 **"나 누군지(어떤 쉘인지) 알려주는 주민등록증"** 입니다.

```bash
#!/bin/bash
# 👆 'Shebang(#!)'이라고 부르며, /bin/bash로 실행하라는 뜻입니다.

# 주석(Comment)은 #으로 시작합니다.
echo "Hello, Linux World!"
```

### 실행 방법 (Permission) 

파일을 만들면 기본적으로 **실행 권한(`x`)** 이 없습니다. 허락을 해줘야 합니다.

```bash
# 1. 실행 권한 부여 (필수!)
chmod +x my_script.sh

# 2. 실행하기 (현재 경로 './'를 꼭 붙여야 함)
./my_script.sh
```

## 핵심 문법 ②: 변수 (Variables) 

데이터를 담는 그릇입니다. **"만들 땐 그냥, 쓸 땐 `$`"** 만 기억하세요.

```bash
#!/bin/bash

# 1. 변수 만들기 (🚨 주의: 등호 앞뒤 공백 절대 금지!)
name="Alice"      # (O)
# name = "Alice"  # (X) 에러 발생!

# 2. 변수 사용하기 ($ 붙이기)
echo "Hello, $name!"
```

## 핵심 문법 ③: 조건문 (If / Else) 

"오전이면 굿모닝, 아니면 굿이브닝" 처럼 상황을 판단합니다.

> **문법 주의:** 대괄호 `[ ]` 안쪽 양옆에는 **무조건 공백**이 있어야 합니다.

- 문법 : `{bash}if [조건]; then ... fi`

```bash
#!/bin/bash

hour=$(date +%H)  # 현재 시간을 변수에 저장

# if [조건]; then ... fi 구조
if [ "$hour" -lt 12 ]; then        # 12보다 작으면 (Less Than)
    echo "Good morning!"
elif [ "$hour" -lt 18 ]; then      # 18보다 작으면
    echo "Good afternoon!"
else                               # 그 외
    echo "Good evening!"
fi # if문을 뒤집은 글자. "여기서 끝!"이라는 뜻
```

### 비교 연산자 치트시트

|**종류**|**기호**|**의미**|**예시**|
|---|---|---|---|
|**숫자**|`-eq`|같다 (Equal)|`[ $a -eq 10 ]`|
||`-ne`|다르다 (Not Equal)|`[ $a -ne 0 ]`|
||`-gt` / `-lt`|초과 / 미만|`[ $a -gt 5 ]`|
||`-ge` / `-le`|이상 / 이하|`[ $a -le 100 ]`|
|**문자열**|`=`|같다|`[ "$str" = "yes" ]`|
||`!=`|다르다|`[ "$str" != "no" ]`|
|**파일**|`-f`|파일이 있는가?|`[ -f "log.txt" ]`|
||`-d`|디렉토리인가?|`[ -d "/tmp" ]`|

## 핵심 문법 ④: 반복문 (Loops) 

### A. For 문 (리스트 반복)

가장 일반적인 방식입니다. 주어진 목록을 하나씩 꺼냅니다.

```bash
# 1부터 5까지 하나씩 i에 넣고 돌리기
for i in 1 2 3 4 5; do
    echo "Number: $i"
done
```

### B. C언어 스타일 For 문 (숫자 카운팅)

**숫자를 카운트할 때** 훨씬 직관적입니다.
- **특징:** 이중 괄호 `(( ... ))`를 사용합니다.
- **장점:** 파이썬의 `range()`나 C/Java 문법과 비슷해서 익숙합니다.

```bash
# i는 1부터; 50보다 작거나 같을 때까지; 1씩 증가시킨다(++)
for ((i=1; i<=50; i++)); do
    echo "Count: $i"
done
```

### C. While 문 (조건 반복)

조건이 맞을 때까지 무한정 반복합니다.

```bash
count=0
while [ "$count" -lt 5 ]; do
    echo "Count: $count"
    count=$((count + 1))  # 숫자 계산은 $(( )) 안에서
done
```

**문법 구조:**

- `if`는 **`fi`** 로 닫고 (finish)
- `for/while`은 **`do`** 로 열고 **`done`** 으로 닫습니다. (실행하라 -> 완료했다)

## 초보자가 자주 하는 실수 (Troubleshooting) 

### ① "변수 선언했는데 `command not found` 에러가 떠요!"

- **원인:** `name = "Alice"` 처럼 띄어쓰기를 했기 때문입니다.
- **해결:** `name="Alice"` (딱 붙여쓰세요!)

### ② "If 문에서 `missing ]` 에러가 나요!"

- **원인:** `if ["$a" -eq 1]` 처럼 대괄호 안을 붙여 썼기 때문입니다.
- **해결:** `if [ "$a" -eq 1 ]` (안쪽에 공백 필수!)

### ③ "`./script.sh` 와 `source script.sh` 차이가 뭐예요?"

- **`./` (실행):** 자식 프로세스(새 창)를 열어서 실행하고 닫습니다. (변수가 증발함)
- **`source` (적용):** 현재 창에 바로 실행합니다. (환경 변수 설정할 때 사용)
