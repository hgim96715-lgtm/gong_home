# ① 정렬 & 패딩 — `:<`, `:>`, `:^`

```
{값:<너비}   왼쪽 정렬 (Left)   나머지는 공백으로 채움
{값:>너비}   오른쪽 정렬 (Right)
{값:^너비}   가운데 정렬 (Center)
```

```python
stn = "서울"

print(f"|{stn:<6}|")   # |서울    |  ← 왼쪽 정렬, 나머지 공백
print(f"|{stn:>6}|")   # |    서울|  ← 오른쪽 정렬
print(f"|{stn:^6}|")   # |  서울  |  ← 가운데 정렬
```

```python
# 공백 대신 다른 문자로 채우기
print(f"|{stn:━<10}|")   # |서울━━━━━━━━|
print(f"|{stn:─^10}|")   # |────서울────|
```

## 실전 — 전광판 출력 (02_API_Producer)

```python
# 실제 코드에서 쓴 패턴
print(
    f" 👉 현재: {item.get('stn_nm', '미확인역'):<4} ({item.get('stop_se_nm', '운행중'):<4}) | "
    f"출발: {real_dep} | 도착: {real_arr}"

```

```
{item.get('stn_nm', '미확인역'):<4}
  ↑                              ↑
  값 (없으면 '미확인역' 기본값)   너비 4칸, 왼쪽 정렬

역명 길이가 달라도 항상 4칸을 차지하므로
여러 줄 출력 시 세로 정렬이 맞아떨어짐

서울  (운행중) | 출발: 14:30 | 도착: 17:45
수원  (지연  ) | 출발: 14:35 | 도착: 18:00
대구광역(운행중) | 출발: 14:50 | 도착: 17:30
↑ 4칸으로 맞춰져서 가독성이 좋아짐
```

# ② 멀티라인 f-string 이어붙이기

```python
# 방법 1: 괄호 안에서 f-string 두 개 나란히 (자동 이어붙임)
print(
    f" 👉 현재: {stn_nm:<4} ({stop_se:<4}) | "
    f"출발: {real_dep} | 도착: {real_arr}"
)
# ↑ 두 f-string 이 공백 없이 하나로 합쳐짐
```

```
왜 이렇게 쓰나?
  한 줄이 너무 길어지면 가독성이 떨어짐
  파이썬은 괄호 () 안에서 줄바꿈을 허용함
  f"..." f"..." 를 연속으로 쓰면 자동으로 이어붙여짐
  \ (백슬래시) 없이 깔끔하게 줄바꿈 가능

주의: f"..." f"..." 사이에는 아무것도 없어야 함
      f"..." + f"..." 이 아님 (+ 없음)
```

```python
# 방법 2: 변수에 담아서 합치기
line = (
    f" 👉 현재: {stn_nm:<4} ({stop_se:<4}) | "
    f"출발: {real_dep} | 도착: {real_arr}"
)
print(line)

# 방법 3: 백슬래시 (비추천 — 가독성 나쁨)
line = f" 👉 현재: {stn_nm:<4} " \
       f"출발: {real_dep}"
```

# ③ 숫자 포맷

```python
n     = 1234567
ratio = 0.87654

# 천 단위 구분자
print(f"{n:,}")          # 1,234,567
print(f"{n:_}")          # 1_234_567  (파이썬 3.6+)

# 소수점 자릿수
print(f"{ratio:.2f}")    # 0.88   ← 소수점 2자리, 반올림
print(f"{ratio:.4f}")    # 0.8765

# 퍼센트
print(f"{ratio:.1%}")    # 87.7%

# 정수 + 천 단위
price = 15000
print(f"가격: {price:,}원")   # 가격: 15,000원

# 자릿수 + 정렬 조합
print(f"{n:>10,}")       #  1,234,567  ← 10칸 오른쪽 정렬
print(f"{n:0>10}")       # 0001234567  ← 빈 자리를 0 으로 채움
```

# ④ 날짜 포맷 (datetime 과 함께)

```python
from datetime import datetime
now = datetime.now()

# f-string 안에서 strftime 포맷 직접 지정
print(f"{now:%Y-%m-%d}")           # 2026-02-24
print(f"{now:%Y-%m-%d %H:%M}")     # 2026-02-24 21:06
print(f"수집 시각: {now:%H:%M:%S}")  # 수집 시각: 21:06:32
```

```
f-string 안에서 : 뒤에 strftime 포맷 코드를 바로 쓸 수 있음
datetime.strftime() 을 따로 호출하지 않아도 됨
```

# ⑤ 중첩 표현식 & 조건식

```python
value = None

# 기본값 처리 (or 패턴)
print(f"역명: {value or '미확인'}")    # 역명: 미확인

# 삼항 연산자
delay = 5
print(f"상태: {'지연' if delay > 0 else '정상'}")   # 상태: 지연

# .get() 으로 None 방어
item = {"stn_nm": "수원", "stop_se_nm": None}
print(f"{item.get('stn_nm', '미확인'):<4} | {item.get('stop_se_nm') or '운행중':<4}")
# 수원   | 운행중
```

# 포맷 지정자 치트시트

|포맷|의미|예시|
|---|---|---|
|`:<N`|왼쪽 정렬, N칸|`{'서울':<6}` → `'서울 '`|
|`:>N`|오른쪽 정렬, N칸|`{42:>6}` → `' 42'`|
|`:^N`|가운데 정렬, N칸|`{'A':^5}` → `' A '`|
|`:0>N`|0 으로 빈 칸 채움|`{7:0>3}` → `'007'`|
|`:.Nf`|소수점 N자리|`{3.14:.1f}` → `'3.1'`|
|`:,`|천 단위 구분자|`{1000:,}` → `'1,000'`|
|`:.N%`|퍼센트 N자리|`{0.876:.1%}` → `'87.6%'`|
|`:%Y-%m-%d`|날짜 포맷|`{now:%Y-%m-%d}`|

# 초보자 착각 포인트

```
① {값:<4} 에서 < 와 숫자 사이에 공백 없음
   {stn:<4}  ← 맞음
   {stn: <4} ← 공백 있으면 채우는 문자가 공백으로 인식됨 (결과는 같지만 헷갈림)

② f"..." f"..." 이어붙이기에서 + 넣으면 안 됨
   f"A" f"B"   → "AB"  ← 맞음
   f"A" + f"B" → "AB"  ← 되긴 하지만 불필요한 + 연산

③ 숫자 너비와 문자 너비 차이
   한글 한 글자는 터미널에서 2칸을 차지함
   :<4 로 맞춰도 한글이 섞이면 완벽히 정렬이 안 될 수 있음
   → 전광판 출력처럼 정밀한 정렬이 필요하면 실제 출력 확인 필수
```
