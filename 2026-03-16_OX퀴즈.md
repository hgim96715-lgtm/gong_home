## 문제 분석 & 설계 (Code Before Think)

> 덧셈, 뺄셈 수식들이 'X [연산자] Y = Z' 형태로 들어있는 문자열 배열 `quiz`가 매개변수로 주어집니다. 
> 수식이 옳다면 "O"를 틀리다면 "X"를 순서대로 담은 배열을 return하도록 solution 함수를 완성해주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: `quiz` : `"3 - 4 = -1"` 같은 **수식 문자열들이 들어있는 배열(list)**
    - **Output**: 각 수식의 정답 여부를 판별한 **`"O"` 또는 `"X"`가 담긴 배열(list)**
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)

3. **사용할 함수/라이브러리:**
   - `split()`
   - `append()`

## ✅ 정답 코드 (Solution)

```python
def solution(quiz):
    answer = []
    for q in quiz:
        q=q.split()
        a,op,b,_,c=q
        res= int(a) + int(b) if op == '+' else int(a) - int(b)
        # print(result)
        answer.append('X' if res!=int(c) else 'O')
    return answer
```
