## 문제 분석 & 설계 (Code Before Think)

> 두 정수 a, b가 주어졌을 때 a와 b 사이에 속한 모든 정수의 합을 리턴하는 함수, solution을 완성하세요.  
> 예를 들어 a = 3, b = 5인 경우, 3 + 4 + 5 = 12이므로 12를 리턴합니다.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 두 정수 `a`, `b`
    - **Output**: `a`와 `b` 사이 (포함)의 모든 정수의 합
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- `range(start, end+1)`을 이용해 구간 생성
3. **사용할 함수/라이브러리:**
   - `range()` : 연속된 정수 생성

---
## ✅ 정답 코드 (Solution)

```python
def solution(a, b):
    answer =0
    if a>b:
        answer=sum([n for n in range(b,a+1)])
    else:
        answer=sum([n for n in range(a,b+1)])
    return answer
```

---
## 오답 노트 & 배운 점 (Retrospective)

-  **내가 실수한 부분 (Syntax/Logic):**
    
-  **새로 알게 된 함수/메서드:**
	-  sum() 은 리스트뿐 아니라 iterable 이면 뭐든 받음 
	- range() 는 iterable → `[n for n in range()]` 굳이 할 필요 없음!!!

## 더 나은 풀이 (Pythonic Code)

> 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
> min, max를 이용하거나 swap을 이용해 하는것 !!


```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def adder(a, b):
    if a > b:
        a, b = b, a
    return sum(range(a, b + 1))

# 아래는 테스트로 출력해 보기 위한 코드입니다.
print( adder(3, 5))



def solution(a, b):
    return sum(range(min(a, b), max(a, b) + 1))
```
