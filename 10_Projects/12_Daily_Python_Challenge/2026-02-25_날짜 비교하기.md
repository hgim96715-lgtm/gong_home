## 문제 분석 & 설계 (Code Before Think)

> 정수 배열 `date1`과 `date2`가 주어집니다. 두 배열은 각각 날짜를 나타내며 `[year, month, day]` 꼴로 주어집니다. 각 배열에서 `year`는 연도를, `month`는 월을, `day`는 날짜를 나타냅니다.
> 만약 `date1`이 `date2`보다 앞서는 날짜라면 1을, 아니면 0을 return 하는 solution 함수를 완성해 주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 정수 배열 `date1`과 `date2`
    - **Output**: `date1`이 `date2`보다 앞서는 날짜라면 1을, 아니면 0
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)

3. **사용할 함수/라이브러리:**
   - `zip`

---
## ✅ 정답 코드 (Solution)

```python
def solution(date1, date2):
    for a, b in zip(date1, date2):
        if a-b<0:
            return 1
        elif a==b:
            continue
        else:
            return 0
    return 0
```

## 오답 노트 & 배운 점 (Retrospective)

-  **내가 실수한 부분 (Syntax/Logic):**
	- 처음에 비교할때 `a == b` → **아직 판단 불가 → 다음 자리(year → month → day)로 이동** 해야하는데 바로 return을 했었음
    
-  **새로 알게 된 함수/메서드:**
	- 파이썬은 리스트를 **앞에서부터 요소 하나씩 비교**
	- 첫 번째로 다른 값이 나오면 그 값으로 전체 비교 결과 결정
	- 완전히 같으면 `False`
