## 문제 분석 & 설계 (Code Before Think)

> 양의 정수 `n`이 매개변수로 주어질 때, `n`이 홀수라면 `n` 이하의 홀수인 모든 양의 정수의 합을 return 하고 `n`이 짝수라면 `n` 이하의 짝수인 모든 양의 정수의 제곱의 합을 return 하는 solution 함수를 작성해 주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
	- 입력 (Input): 정수 `n` (양의 정수)
	- 출력 (Output) : `n`이 **홀수**일 때->1부터 n까지의 홀수의 합, `n`이 **짝수**일 때-> 1부터 n까지의 짝수의 제곱의 합
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- `n % 2 == 1` → 홀수 처리, `n % 2 == 0` → 짝수 처리
3. **사용할 함수/라이브러리:**
   - range(start, end, step)
   - `i ** 2` (제곱)

---
## ✅ 정답 코드 (Solution)

```python
def solution(n):
    answer = 0
    if n%2==1:
        for i in range(n+1):
            if i%2==1:
                answer+=i
            else: 
                answer+=0
    else:
        for i in range(1,n+1):
            if i%2==0:
                answer+=i**2
    return answer
```

## 더 나은 풀이 (Pythonic Code)

> 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?


```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def solution(n):
    if n%2:
        return sum(range(1,n+1,2))
    return sum([i*i for i in range(2,n+1,2)])
```

- 홀수만 필요 → `range(1, n+1, 2)`
- 짝수만 필요 → `range(2, n+1, 2)`

> `if n % 2:` 는 `n`이 홀수일 때만 True가 된다

```python
if n % 2:
    print("홀수")
else:
    print("짝수")
```
