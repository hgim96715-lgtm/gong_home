## 문제 분석 & 설계 (Code Before Think)

> 정수 배열 `arr`과 정수 `n`이 매개변수로 주어집니다. 
> `arr`의 길이가 홀수라면 `arr`의 모든 짝수 인덱스 위치에 `n`을 더한 배열을, `arr`의 길이가 짝수라면 `arr`의 모든 홀수 인덱스 위치에 `n`을 더한 배열을 return 하는 solution 함수를 작성해 주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**:  정수 배열 `arr`과 정수 `n`
    - **Output**:  `arr` 길이 홀수 : 짝수 인덱스 위치 + n , `arr` 길이 짝수 : 홀수 인덱스위치 +n
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- 길이로 if -else
3. **사용할 함수/라이브러리:**
	- `len`

## ✅ 정답 코드 (Solution)

```python
def solution(arr, n):
    for i,v in enumerate(arr):
        if len(arr)%2:
            if i%2==0:
                 arr[i]+=n
        else:
            if i%2!=0:
                arr[i]+=n
    return arr
```

```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def solution(arr, n):
    N=len(arr)
    if N%2:
        for i in range(0,N,2): arr[i]+=n
    else:
        for i in range(1,N,2): arr[i]+=n
    return arr
```
