## 문제 분석 & 설계 (Code Before Think)

> 정수를 담고 있는 배열 arr의 평균값을 return하는 함수, solution을 완성해보세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 정수들이 담긴 배열 `arr`
    - **Output**: 배열 원소들의 평균값 (float)
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- 평균 = 총합 / 개수
3. **사용할 함수/라이브러리:**
   - `sum()` : 배열의 합 계산
   - `len()` : 배열의 길이 계산

---
## ✅ 정답 코드 (Solution)

```python
def solution(arr):
    answer = sum(a for a in arr)/len(arr)
    return answer
```
