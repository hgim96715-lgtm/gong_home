## 문제 분석 & 설계 (Code Before Think)

> 문자열 `my_string`, `overwrite_string`과 정수 `s`가 주어집니다. 
> 문자열 `my_string`의 인덱스 `s`부터 `overwrite_string`의 길이만큼을 문자열 `overwrite_string`으로 바꾼 문자열을 return 하는 solution 함수를 작성해 주세요.

		- `overwrite_string` : 덮어쓸 문자열
		- `s` : 덮어쓰기를 시작할 인덱스 (정수)
	- 출력

## ✅ 정답 코드 (Solution)

```python
def solution(my_string, overwrite_string, s):
   return f"{my_string[:s]}{overwrite_string}{my_string[s+len(overwrite_string):]}"
```

## 오답 노트 & 배운 점 (Retrospective)

-  **내가 실수한 부분 (Syntax/Logic):**
    - `replace()`는 **"위치(Index)"가 아니라 "값(Value)"** 을 찾아서 바꾼다.
    - 파이썬의 문자열(String)은 불변(Immutable) 객체
-  **새로 알게 된 함수/메서드:**
	- 기존 문자열을 수정하는 것이 아니라 **새 문자열을 생성**해야 함
	- 문자열 연결 (`+`)→ 여러 문자열을 하나의 문자열로 결합
