## 문제 분석 & 설계 (Code Before Think)

> 양의 정수 `x`가 하샤드 수이려면 `x`의 자릿수의 합으로 `x`가 나누어져야 합니다. 예를 들어 18의 자릿수 합은 1+8=9이고, 18은 9로 나누어 떨어지므로 18은 하샤드 수입니다. 자연수 `x`를 입력받아 `x`가 하샤드 수인지 아닌지 검사하는 함수, solution을 완성해주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 자연수 `x`
    - **Output**: 하샤드 수이면 `True`  아니면 `False`
 2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- 자릿수 합 → 나눗셈 → 나머지 체크
3. **사용할 함수/라이브러리:**
   - `str()` : 숫자를 문자열로 변환
   - `list()` : 자릿수 분리

---
## ✅ 정답 코드 (Solution)

```python
def solution(x):
    sum_arr=sum(int(a) for a in str(x))
    return True if x%sum_arr==0 else False
```

# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def Harshad(n):
    return n%sum(int(x) for x in str(n)) == 0

# 아래는 테스트로 출력해 보기 위한 코드입니다.
print(Harshad(18))
