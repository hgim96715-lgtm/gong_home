## ✅ 정답 코드 (Solution)

```python
def solution(myString, pat):
    answer = 0
    for i,v in enumerate(myString):
        p=myString[i:i+len(pat)]
        if p == pat:
            answer+=1
    return answer
```

## 오답 노트 & 배운 점 (Retrospective)

-  **내가 실수한 부분 (Syntax/Logic):**
	- 문자열 비교에서 `p == pat`로 **완전 일치**를 확인해야 하는데 `p in pat`을 사용해 **부분 문자열 포함 여부**로 잘못 판단
    
-  **새로 알게 된 함수/메서드:**
	- `in` 연산자는 **부분 포함 여부**를 검사할 때 사용하며 문자열 등장 횟수를 셀 때는 **슬라이싱 + 동등 비교(`{text}==`)** 가 더 정확

>`"등장 횟수"` 문제에서는 `in`이 아니라 **`{text}==`가 정답이다.**
