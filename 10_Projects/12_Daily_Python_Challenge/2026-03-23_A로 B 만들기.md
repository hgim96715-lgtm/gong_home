## 문제 분석 & 설계 (Code Before Think)

> 문자열 `before`와 `after`가 매개변수로 주어질 때, `before`의 순서를 바꾸어 `after`를 만들 수 있으면 1을, 만들 수 없으면 0을 return 하도록 solution 함수를 완성해보세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 문자열 `before` , 문자열 `after`
    - **Output**: `before`의 문자 순서를 바꿔 `after`를 만들 수 있으면 `1` , 아니면 `0`
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)

3. **사용할 함수/라이브러리:**
   - `sorted()` : 문자열을 정렬하여 리스트로 반환

---
## ✅ 정답 코드 (Solution)

```python
def solution(before, after):
    return 1 if sorted(before)==sorted(after) else 0
```

## 더 나은 풀이 (Pythonic Code)

> 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?


```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
from collections import Counter

def solution(before, after):
    return int(Counter(before) == Counter(after))
```
