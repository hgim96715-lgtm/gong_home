## 문제 분석 & 설계 (Code Before Think)

> 단어 s의 가운데 글자를 반환하는 함수, solution을 만들어 보세요. 단어의 길이가 짝수라면 가운데 두글자를 반환하면 됩니다.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 문자열 `s`
    - **Output**: 문자열 (가운데 문자 또는 가운데 두 문자)
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- 문자열 길이를 구한다 → `len(s)`
	- 길이가 **홀수**면:`len(s)//2` 
	- 길이가 **짝수**면:`len(s)//2 - 1` , `len(s)//2`
3. **사용할 함수/라이브러리:**
   - 슬라이싱 `s[start:end]`
   - `len()` : 문자열 길이

---
## ✅ 정답 코드 (Solution)

```python
def solution(s):
    answer = s[len(s)//2] if len(s)%2 else s[len(s)//2-1:len(s)//2+1]
    return answer
```

# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def string_middle(str):
    return str[(len(str)-1)//2 : len(str)//2 + 1]
