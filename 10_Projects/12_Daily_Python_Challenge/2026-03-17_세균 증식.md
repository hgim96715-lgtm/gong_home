## 문제 분석 & 설계 (Code Before Think)

> 어떤 세균은 1시간에 두배만큼 증식한다고 합니다. 
> 처음 세균의 마리수 `n`과 경과한 시간 `t`가 매개변수로 주어질 때 `t`시간 후 세균의 수를 return하도록 solution 함수를 완성해주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: `n` : 처음 세균의 마리수 , `t` : 경과한 시간
    - **Output**: `t`시간 후의 세균의 마리수
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- 초기값 `n`에서 시작하여 **t번 반복하면서 2배씩 증가**
3. **사용할 함수/라이브러리:**

## ✅ 정답 코드 (Solution)

```python
def solution(n, t):
    answer = n
    for a in range(1,t+1):
        answer*=2
    return answer
```
