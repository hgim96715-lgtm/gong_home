## 문제 분석 & 설계 (Code Before Think)

>두 수를 입력받아 두 수의 최대공약수와 최소공배수를 반환하는 함수, solution을 완성해 보세요. 배열의 맨 앞에 최대공약수, 그다음 최소공배수를 넣어 반환하면 됩니다. 예를 들어 두 수 3, 12의 최대공약수는 3, 최소공배수는 12이므로 solution(3, 12)는 [3, 12]를 반환해야 합니다.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 두 개의 정수 `n`, `m`
    - **Output**: `[최대공약수, 최소공배수]` 형태의 리스트
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- 먼저 두 수의 **최대공약수(GCD)** 를 구한다.
	- 최소공배수(LCM)는 `두 수의 곱 ÷ 최대공약수` 공식을 이용해 구한다
3. **사용할 함수/라이브러리:**
    - `math.gcd()` : 최대공약수 계산
    - 정수 나눗셈 연산자 `//`

---
## ✅ 정답 코드 (Solution)

```python
import math
def solution(n, m):
    ma_gc=math.gcd(n,m)
    ma_lc=(n*m)//ma_gc
    answer = [ma_gc,ma_lc]
    return answer
```
