## 문제 분석 & 설계 (Code Before Think)

> 약수의 개수가 세 개 이상인 수를 합성수라고 합니다. 자연수 `n`이 매개변수로 주어질 때 `n`이하의 합성수의 개수를 return하도록 solution 함수를 완성해주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 자연수 `n`
    - **Output**: `n` 이하의 **합성수의 개수**
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)

3. **사용할 함수/라이브러리:**
   - `collections.Counter` : 리스트에 들어있는 값의 **등장 횟수(약수 개수)** 를 계산하기 위해 사용

---
## ✅ 정답 코드 (Solution)

```python
from collections import Counter
def solution(n):
    answer = []
    arr=[]
    for i in range(1,n+1):
        for j in range(1,i+1):
            if i%j==0:
                answer.append(i)
    counter=Counter(answer)
    for v,c in counter.items():
        if c>=3:
            arr.append(v)
    return len(arr)
```

```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def solution(n):
    answer = 0
    for i in range(3,n+1) :
        temp = 1
        for j in range(1,i) :
            if i % j == 0 :
                temp += 1

        if temp > 2 :
            answer += 1
    return answer
```
