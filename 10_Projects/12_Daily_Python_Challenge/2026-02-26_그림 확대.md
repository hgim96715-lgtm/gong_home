## 문제 분석 & 설계 (Code Before Think)

> 직사각형 형태의 그림 파일이 있고, 이 그림 파일은 1 × 1 크기의 정사각형 크기의 픽셀로 이루어져 있습니다. 이 그림 파일을 나타낸 문자열 배열 `picture`과 정수 `k`가 매개변수로 주어질 때, 이 그림 파일을 가로 세로로 `k`배 늘린 그림 파일을 나타내도록 문자열 배열을 return 하는 solution 함수를 작성해 주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 문자열 배열 `picture`과 정수 `k`
    - **Output**: 문자열 배열
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)

3. **사용할 함수/라이브러리:**
   - `for`
   - `append`

---
## ✅ 정답 코드 (Solution)

```python
def solution(picture, k):
    answer =[]
    for p in picture:
        ch=''
        for r in p:
            ch+=r*k
        # print(ch)
        for _ in range(k):
            answer.append(ch)
    return answer
```

---
## 오답 노트 & 배운 점 (Retrospective)

-  **내가 실수한 부분 (Syntax/Logic):**
	- 처음에는 `p * k` 를 사용해 문자열을 확장하려 했는데,이는 문자열 전체를 반복하는 것이지 문자 하나씩 반복하는 것이 아니었다.
    
-  **새로 알게 된 함수/메서드:**
	- `문자열 * k` 는 문자열 전체를 반복
	- `replace()` 메서드를 활용하면 특정 문자를 원하는 문자열로 한 번에 치환

```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def solution(picture, k):
    answer = []
    for i in range(len(picture)):
        for _ in range(k):
            answer.append(picture[i].replace('.', '.' * k).replace('x', 'x' * k))
    return answer
```
