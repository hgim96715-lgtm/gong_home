## 문제 분석 & 설계 (Code Before Think)

> 문자열 s의 길이가 4 혹은 6이고, 숫자로만 구성돼있는지 확인해주는 함수, solution을 완성하세요. 
> 예를 들어 s가 "a234"이면 False를 리턴하고 "1234"라면 True를 리턴하면 됩니다.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 문자열 `s` (길이 제한 없음, 문자/숫자 혼합 가능)
    - **Output**: `bool` (조건 만족 시 `True`, 아니면 `False`)
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- 문자열의 길이가 **4 또는 6인지 확인**
	- 문자열이 **숫자로만 이루어져 있는지 확인**
3. **사용할 함수/라이브러리:**
	-  `str.isdigit()` : 문자열이 숫자로만 구성되어 있는지 확인
    -  `len()` : 문자열 길이 확인

---
## ✅ 정답 코드 (Solution)

```python
def solution(s):
    answer = True if s.isdigit() and ( len(s)==4 or len(s)==6) else False
    return answer
```

	- `len(s) in [4,6]`처럼 **멤버십 검사로 조건을 더 간결하게 표현 가능**
