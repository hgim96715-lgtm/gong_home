## 문제 분석 & 설계 (Code Before Think)

> 정수 배열 `arr`이 매개변수로 주어집니다. `arr`의 길이가 2의 정수 거듭제곱이 되도록 `arr` 뒤에 정수 0을 추가하려고 합니다. 
> `arr`에 최소한의 개수로 0을 추가한 배열을 return 하는 solution 함수를 작성해 주세요.
> 제한사항
> 1 ≤ `arr`의 길이 ≤ 1,000
>  1 ≤ `arr`의 원소 ≤ 1,000

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 정수 배열 `arr`
    - **Output**: `arr`에 최소한의 개수로 0을 추가한 배열
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- `while` 반복문을 사용해 조건을 만족할 때까지 점진적으로 0을 추가
	- 2의 거듭제곱 수들을 미리 리스트로 만든다.
3. **사용할 함수/라이브러리:**
   - `pow(2, i)` : 2의 거듭제곱 값을 생성

---
## ✅ 정답 코드 (Solution)

```python
def solution(arr):
    answer = []
    pow_2=[]
    for i in range(1000):
        pow_2+=[pow(2,i)]
    while len(arr) not in pow_2:
        arr.append(0)
        
    return arr
```

# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def solution(arr):
    length = len(arr)
    target = 1 << (length - 1).bit_length()
    return arr + [0] * (target - length)
