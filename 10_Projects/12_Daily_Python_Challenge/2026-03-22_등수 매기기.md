## 문제 분석 & 설계 (Code Before Think)

> 영어 점수와 수학 점수의 평균 점수를 기준으로 학생들의 등수를 매기려고 합니다. 영어 점수와 수학 점수를 담은 2차원 정수 배열 `score`가 주어질 때, 영어 점수와 수학 점수의 평균을 기준으로 매긴 등수를 담은 배열을 return하도록 solution 함수를 완성해주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: `score` → 2차원 리스트`[영어점수, 수학점수]`
    - **Output**: 각 학생의 평균 점수를 기준으로 매긴 **등수 리스트**
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)

3. **사용할 함수/라이브러리:**
   - `list.index()` → 값의 위치 찾기

## ✅ 정답 코드 (Solution)

```python
def solution(score):
    arr=[]
    answer=[]
    for s in score:
        arr.append((s[0] + s[1]) / 2)
    arr_sort=sorted(arr,reverse=True)
    for v in arr:
        answer.append(arr_sort.index(v) + 1)
    return answer
```

```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def solution(score):
    rank = sorted([sum(s) / 2 for s in score], reverse=True)
    rankDict = {}
    for i, r in enumerate(rank):
        if r not in rankDict.keys():
            rankDict[r] = i + 1
    return [rankDict[sum(s) / 2] for s in score]
```
