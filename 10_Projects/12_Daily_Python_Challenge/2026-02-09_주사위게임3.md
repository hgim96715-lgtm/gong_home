## 문제 분석 & 설계 (Code Before Think)

>1부터 6까지 숫자가 적힌 주사위가 네 개 있습니다. 
>네 주사위를 굴렸을 때 나온 숫자에 따라 다음과 같은 점수를 얻습니다.
> 네 주사위에서 나온 숫자가 모두 p로 같다면 1111 × p점을 얻습니다.
>세 주사위에서 나온 숫자가 p로 같고 나머지 다른 주사위에서 나온 숫자가 q(p ≠ q)라면 (10 × p + q)2 점을 얻습니다.
>주사위가 두 개씩 같은 값이 나오고, 나온 숫자를 각각 p, q(p ≠ q)라고 한다면 (p + q) × |p - q|점을 얻습니다.
>어느 두 주사위에서 나온 숫자가 p로 같고 나머지 두 주사위에서 나온 숫자가 각각 p와 다른 q, r(q ≠ r)이라면 q × r점을 얻습니다.
>네 주사위에 적힌 숫자가 모두 다르다면 나온 숫자 중 가장 작은 숫자 만큼의 점수를 얻습니다

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 정수 매개변수 `a`, `b`, `c`, `d` , 각 값은 1 이상 6 이하의 정수
    - **Output**: 얻는 점수를 return 
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- 중복을 제거 하여 그 안에 배열 개수로 if문을 짜서 로직을 구성 
3. **사용할 함수/라이브러리:**
   - `Set`
   - `list.count()` : 각 숫자의 등장 횟수 확인
   - `min()` : 모두 다른 경우 최솟값 계산

## ✅ 정답 코드 (Solution)

```python
def solution(a, b, c, d):
    arr=[a,b,c,d]
    answer = list(set(arr))
    if len(answer)==1:
        return 1111*answer[0]
    elif len(answer)==4:
        return min(answer)
    elif len(answer)==2:
        p,q=answer[0],answer[1]
        p_c, q_c = arr.count(p), arr.count(q)
        if p_c==3:
            return (10*p+q)**2
        elif q_c==3:
            return (10*q+p)**2
        else:
            return (p + q) * abs(p - q)
    elif len(answer)==3:
        one=[v for v in answer if arr.count(v)==1]
        # print(one)
        return one[0]*one[1]
    return answer
```

## 오답 노트 & 배운 점 (Retrospective)

-  **내가 실수한 부분 (Syntax/Logic):**
	- 처음에 `set`으로 만든 리스트(`answer`)의 순서를 신뢰하려고 했기때문에, 최종 실패가 계속 발생함
	- `set`은 순서를 보장하지 않는다!!!
	- `count()`로 등장 횟수를 확인해서 역할을 정해야 한다는 점을 놓쳤음
-  **새로 알게 된 함수/메서드:**
	- `list.count(x)` : 리스트 안에서 해당 값이 **몇 번 등장하는지** 반환

## 더 나은 풀이 (Pythonic Code)

> 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?


```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def solution(a, b, c, d):
    l = [a,b,c,d]
    c = [l.count(x) for x in l]
    if max(c) == 4:
        return 1111*a
    elif max(c) == 3:
        return (10*l[c.index(3)]+l[c.index(1)])**2
    elif max(c) == 2:
        if min(c) == 1:
            return eval('*'.join([str(l[i]) for i, x in enumerate(c) if x == 1]))
        else:
            return (max(l) + min(l)) * abs(max(l) - min(l))
    else:
        return min(l)
```
