## 문제 분석 & 설계 (Code Before Think)

> 외과의사 머쓱이는 응급실에 온 환자의 응급도를 기준으로 진료 순서를 정하려고 합니다. 
> 정수 배열 `emergency`가 매개변수로 주어질 때 응급도가 높은 순서대로 진료 순서를 정한 배열을 return하도록 solution 함수를 완성해주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 정수 리스트 `emergency`
    - **Output**: 각 환자의 **진료 순서가 담긴 리스트**
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)

3. **사용할 함수/라이브러리:**
   - `sorted()` : 리스트를 **내림차순 정렬**

---
## ✅ 정답 코드 (Solution)

```python
def solution(emergency):
    answer = []
    copy_e=sorted(emergency,reverse=True)
    for e in emergency:
        b=copy_e.index(e)
        answer.append(b+1)
    return answer
```

## 오답 노트 & 배운 점 (Retrospective)

-  **내가 실수한 부분 (Syntax/Logic):**
	- 응급도를 내림차순으로 정렬하는 것까지는 생각했지만,  **정렬된 리스트에서 현재 값의 순위를 찾는 방법을 떠올리지 못했다.**
    
-  **새로 알게 된 함수/메서드:**
	- `list.index()`  → 리스트에서 **특정 값의 위치(인덱스)** 를 찾는 메서드
