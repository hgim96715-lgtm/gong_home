## 문제 분석 & 설계 (Code Before Think)

> 길이가 같은 두 1차원 정수 배열 a, b가 매개변수로 주어집니다. a와 b의 [내적](https://en.wikipedia.org/wiki/Dot_product)을 return 하도록 solution 함수를 완성해주세요.
> 이때, a와 b의 내적은 `a[0]*b[0] + a[1]*b[1] + ... + a[n-1]*b[n-1]` 입니다. (n은 a, b의 길이)

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 정수 배열 `a` (길이 n), 정수 배열 `b` (길이 n) → 두 배열의 길이는 항상 같음
    - **Output**: 두 배열의 내적 결과 (정수)
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- `zip(a, b)`를 사용하면 두 배열을 동시에 순회 가능
3. **사용할 함수/라이브러리:**
   - `zip()` → 두 iterable을 묶어서 동시에 순회

---
## ✅ 정답 코드 (Solution)

```python
def solution(a, b):
    answer = 0
    for n, p in zip(a, b):
        answer+=(n*p)
    return answer
```

## 더 나은 풀이 (Pythonic Code)

> 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?


```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def solution(a, b):
    return sum([x*y for x, y in zip(a,b)])
```
