## 문제 분석 & 설계 (Code Before Think)

> 자연수 n이 매개변수로 주어집니다. n을 3진법 상에서 앞뒤로 뒤집은 후, 이를 다시 10진법으로 표현한 수를 return 하도록 solution 함수를 완성해주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 자연수 `n`
    - **Output**: `n`을 3진법으로 변환 → 뒤집기 → 다시 10진수로 변환한 정
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- 뒤집힌 3진수 문자열을 `int(문자열, 3)`으로 10진수 변환
3. **사용할 함수/라이브러리:**
   - 문자열 슬라이싱 `[:​:-1]`
   - `str()` : 숫자를 문자열로 변환

---
## ✅ 정답 코드 (Solution)

```python
def solution(n):
    answer = 0
    res=''
    while n>0:
        res=str(n%3)+res
        n//=3
    res=res[::-1]
    answer=int(res,3)
    return answer
```
