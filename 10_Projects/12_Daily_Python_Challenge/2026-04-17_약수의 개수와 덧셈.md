## 문제 분석 & 설계 (Code Before Think)

> 두 정수 `left`와 `right`가 매개변수로 주어집니다. `left`부터 `right`까지의 모든 수들 중에서, 약수의 개수가 짝수인 수는 더하고, 약수의 개수가 홀수인 수는 뺀 수를 return 하도록 solution 함수를 완성해주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 정수 `left`, `right` (범위의 시작과 끝)
    - **Output**: 조건에 따라 더하고 뺀 결과값
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- **완전제곱수**만 약수 개수가 **홀수**
3. **사용할 함수/라이브러리:**
   - `math.sqrt()` : 제곱근 계산

---
## ✅ 정답 코드 (Solution)

```python
import math
def solution(left, right):
    answer = 0
    for k in range(left,right+1):
        if  math.sqrt(k) % 1 == 0:
            answer-=k
        else:
            answer+=k
    return answer
```
