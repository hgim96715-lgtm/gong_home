## 문제 분석 & 설계 (Code Before Think)

> 자연수 N이 주어지면, N의 각 자릿수의 합을 구해서 return 하는 solution 함수를 만들어 주세요.  \
> 예를들어 N = 123이면 1 + 2 + 3 = 6을 return 하면 됩니다.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 자연수 N
    - **Output**: 각 자릿수의 합 (정수)
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- `str(n)`으로 변환해서 각 자릿수에 접근한다.
3. **사용할 함수/라이브러리:**
   - `sum()` → 전체 합 계산

---
## ✅ 정답 코드 (Solution)

```python
def solution(n):
    answer = 0
    arr=list(str(n))
    answer=sum(int(a) for a in arr )

    return answer
```

## 더 나은 풀이 (Pythonic Code)

> 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?


```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def sum_digit(number):
    return sum([int(i) for i in str(number)])
print("결과 : {}".format(sum_digit(123)));
```
