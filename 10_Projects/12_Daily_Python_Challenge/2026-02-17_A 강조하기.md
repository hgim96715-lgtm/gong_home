## 문제 분석 & 설계 (Code Before Think)

> 문자열 `myString`이 주어집니다. `myString`에서 알파벳 "a"가 등장하면 전부 "A"로 변환하고, "A"가 아닌 모든 대문자 알파벳은 소문자 알파벳으로 변환하여 return 하는 solution 함수를 완성하세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: myString
    - **Output**:`"a"` 또는 `"A"` → `"A"`로 변환 , `"A"`가 아닌 모든 **대문자 알파벳** → 소문자로 변환
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- 문자열을 **한 글자씩 순회**한다
3. **사용할 함수/라이브러리:**
   - `upper()` : 소문자를 대문자로 변환
   - `lower()` : 대문자를 소문자로 변환

---
## ✅ 정답 코드 (Solution)

```python
def solution(myString):
    answer = ''
    for v in myString:
        if v=='a' or v=='A':
            answer+=v.upper()
        else:
            answer+=v.lower()
    return answer
```
