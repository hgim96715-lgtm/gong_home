## 문제 분석 & 설계 (Code Before Think)

> 문자열 `my_string`이 매개변수로 주어집니다. `my_string`에서 중복된 문자를 제거하고
> 하나의 문자만 남긴 문자열을 return하도록 solution 함수를 완성해주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 문자열 `my_string`
    - **Output**: 중복 문자가 제거된 문자열
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)

3. **사용할 함수/라이브러리:**
   - `for` : 문자열 순

---
## ✅ 정답 코드 (Solution)

```python
def solution(my_string):
    answer = ''
    for m in my_string:
        if m in answer:
            continue
        else:
            answer+=m
    return answer
```

## 오답 노트 & 배운 점 (Retrospective)

-  **내가 실수한 부분 (Syntax/Logic):**
    - 처음에는 `set()`을 사용해서 문자열의 **중복을 제거하려고 했다.**  
    - 하지만 `set` 자료구조는 **입력된 순서를 보장하지 않기 때문에** 문제에서 요구하는 **문자 등장 순서를 유지할 수 없었다.**  
    - 그래서 문자열을 순회하면서 이미 `answer`에 존재하는 문자인지 확인하고, 없는 경우에만 추가하는 방식으로 해결했다.
-  **새로 알게 된 함수/메서드:**
	- `dict.fromkeys()`를 사용하면 **딕셔너리의 키가 중복되지 않는 특성**과 **파이썬 3.7+에서 딕셔너리가 입력 순서를 유지하는 특징**을 활용해 **순서를 유지하면서 중복을 제거**할 수 있다는 것을 알게 되었다.
