## 문제 분석 & 설계 (Code Before Think)

> 정수로 이루어진 문자열 `n_str`이 주어질 때, `n_str`의 가장 왼쪽에 처음으로 등장하는 0들을 뗀 문자열을 return하도록 solution 함수를 완성해주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 정수로 이루어진 문자열 `n_str`
    - **Output**:  0들을 뗀 문자열
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)

3. **사용할 함수/라이브러리:**
   - `startswith`

---
## ✅ 정답 코드 (Solution)

```python
def solution(n_str):
    answer = ''
    while n_str!='' and n_str.startswith('0'):
        n_str=n_str[1:]
    return n_str
```
