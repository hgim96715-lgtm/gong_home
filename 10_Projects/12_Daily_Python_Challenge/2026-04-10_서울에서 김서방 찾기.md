## 문제 분석 & 설계 (Code Before Think)

> String형 배열 seoul의 element중 "Kim"의 위치 x를 찾아, "김서방은 x에 있다"는 String을 반환하는 함수, solution을 완성하세요.
>  seoul에 "Kim"은 오직 한 번만 나타나며 잘못된 값이 입력되는 경우는 없습니다.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 문자열 배열 `seoul` (예: `["Jane", "Kim"]`)
    - **Output**: 문자열 `"김서방은 x에 있다"`
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- Python의 리스트는 **인덱스 기반 접근 가능**
3. **사용할 함수/라이브러리:**
   - `list.index()` 특정값의 위치 반환
   - f-string (`f""`)

---
## ✅ 정답 코드 (Solution)

```python
def solution(seoul):
    kim_in= seoul.index("Kim")
    return f"김서방은 {kim_in}에 있다"
```
