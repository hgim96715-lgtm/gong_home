## 문제 분석 & 설계 (Code Before Think)

> 정수 `n`이 주어질 때, `n`이하의 짝수를 모두 더한 값을 return 하도록 solution 함수를 작성해주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 정수 `n`
    - **Output**:  `n`이하의 짝수를 모두 더한 값
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)

3. **사용할 함수/라이브러리:**
   - `range` 로 2칸씩 이동 해서 짝수의 합을 구하기 

---
## ✅ 정답 코드 (Solution)

```python
def solution(n):
    answer = sum([a for a in range(0,n+1,2)])
    return answer
```

## 더 나은 풀이 (Pythonic Code)

> 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?


>이미 짝수의 합이라고 해서 제한사항에 0초과 였으니깐 2부터 하는게 더 나을 수도 있었겠다고 생각한다.  

```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def solution(n):
    answer = sum([a for a in range(2,n+1,2)])
    return answer
```
