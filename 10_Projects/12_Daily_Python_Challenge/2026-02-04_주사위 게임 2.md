## 문제 분석 & 설계 (Code Before Think)

> 1부터 6까지 숫자가 적힌 주사위가 세 개 있습니다. 
> 세 주사위를 굴렸을 때 나온 숫자를 각각 `a`, `b`, `c`라고 했을 때 얻는 점수는 다음과 같습니다.
> 세 숫자가 모두 다르다면 `a` + `b` + `c` 점을 얻습니다.
>  세 숫자 중 어느 두 숫자는 같고 나머지 다른 숫자는 다르다면 (`a` + `b` + `c`) × (`a`2 + `b`2 + `c`2 )점을 얻습니다.
>  세 숫자가 모두 같다면 (`a` + `b` + `c`) × (`a`2 + `b`2 + `c`2 ) × (`a`3 + `b`3 + `c`3 )점을 얻습니다.
>  세 정수 `a`, `b`, `c`가 매개변수로 주어질 때, 얻는 점수를 return 하는 solution 함수를 작성해 주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 정수 `a`, `b`, `c` (1 이상 6 이하, 주사위 눈)
    - **Output**: 조건에 따라 계산된 **정수 점수**
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- 비교 연산(`{text}==`, `!=`)
3. **사용할 함수/라이브러리:**
	-  기본 산술 연산자 (`+`, `*`, `**`)

## ✅ 정답 코드 (Solution)

```python
def solution(a, b, c):
    answer = 0
    if a!=b and a!=c and b!=c:
        answer=a+b+c
    elif a==b and b==c and a==c:
        answer=(a+b+c)*(a**2+b**2+c**2)*(a**3+b**3+c**3)
    elif a!=b==c or a==b!=c or a==c!=b:
        answer=(a+b+c)*(a**2+b**2+c**2)
    return answer
```

## 더 나은 풀이 (Pythonic Code)

> 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?


```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def solution(a, b, c):
    nums = [a, b, c]
    s = sum(nums)
    
    if len(set(nums)) == 3:        # 모두 다름
        return s
    elif len(set(nums)) == 2:      # 두 개만 같음
        return s * sum(x**2 for x in nums)
    else:                          # 모두 같음
        return s * sum(x**2 for x in nums) * sum(x**3 for x in nums)
```
