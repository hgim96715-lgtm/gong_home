## 문제 분석 & 설계 (Code Before Think)

> 문자열 `my_string`과 정수 `n`이 매개변수로 주어질 때, `my_string`의 뒤의 `n`글자로 이루어진 문자열을 return 하는 solution 함수를 작성해 주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**:  문자열 `my_string`과 정수 `n`
    - **Output**: my_string`의 뒤의 `n`글자로 이루어진 문자열을 return 
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- Python 문자열 슬라이싱을 사용
	- 음수 인덱스를 활용
3. **사용할 함수/라이브러리:**
	- 문자열 슬라이싱 (`[]`)
	- `[: :-1]` (문자열 뒤집기)

---
## ✅ 정답 코드 (Solution)

```python
def solution(my_string, n):
    answer = my_string[-1:-n-1:-1]
    return answer[::-1]
```

## 오답 노트 & 배운 점 (Retrospective)

-  **내가 실수한 부분 (Syntax/Logic):**
    -  `my_string[-1:-n-1]` 처럼 **step을 생략**함
    - Python 슬라이싱은 기본 step이 `+1`이라  
    - **시작 인덱스가 끝 인덱스보다 뒤에 있으면 빈 문자열**이 반환됨
    - 뒤에서 앞으로 슬라이싱할 때는 반드시 `step = -1`이 필요하다는 점을 놓침
