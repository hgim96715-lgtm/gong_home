## 문제 분석 & 설계 (Code Before Think)

> 길이가 n이고, "수박수박수박수...."와 같은 패턴을 유지하는 문자열을 리턴하는 함수, solution을 완성하세요. 
> 예를들어 n이 4이면 "수박수박"을 리턴하고 3이라면 "수박수"를 리턴하면 됩니다.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 정수 `n` (문자열 길이)
    - **Output**: 문자열 (`"수박수박..."` 패턴, 길이 n)
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- `"수박"` 문자열을 반복하면 `"수박수박수박..."` 형태 생성
	- 최소 `n` 길이 이상이 되도록 반복
3. **사용할 함수/라이브러리:**
   - 문자열 곱셈: `"수박" * n`

---
## ✅ 정답 코드 (Solution)

```python
def solution(n):
    answer = ''
    for s in '수박'*n:
        if len(answer)<n:
            answer+=s
    return answer
```

---
## 오답 노트 & 배운 점 (Retrospective)

-  **내가 실수한 부분 (Syntax/Logic):**
	- `len(answer) < n` 조건으로 길이를 계속 체크 → 불필요한 연산
-  **새로 알게 된 함수/메서드:**
	- 최소 `n` 길이 이상이 되도록 반복한 뒤 슬라이싱으로 n 길이만큼 잘라서 반환
