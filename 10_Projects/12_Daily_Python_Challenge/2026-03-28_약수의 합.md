## 문제 분석 & 설계 (Code Before Think)

> 정수 n을 입력받아 n의 약수를 모두 더한 값을 리턴하는 함수, solution을 완성해주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 정수 n
    - **Output**: n의 약수를 모두 더한값 
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- 각 숫자 `i`가 n의 약수인지 확인 (`n % i == 0`)
3. **사용할 함수/라이브러리:**
   - `%` (나머지 연산자) 약수 판별
   - 약수 판별: `n % i == 0`

---
## ✅ 정답 코드 (Solution)

```python
def solution(n):
    answer = 0
    for i in range(1,n+1):
        if n%i==0:
            answer+=i
    return answer
```

# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def sumDivisor(num):
    return sum([i for i in range(1,num+1) if num%i==0])
