## 문제 분석 & 설계 (Code Before Think)

> 문자열 s에 나타나는 문자를 큰것부터 작은 순으로 정렬해 새로운 문자열을 리턴하는 함수, solution을 완성해주세요.  
> s는 영문 대소문자로만 구성되어 있으며, 대문자는 소문자보다 작은 것으로 간주합니다.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 문자열 `s` (영문 대소문자로만 구성)
    - **Output**: 문자를 내림차순으로 정렬한 새로운 문자열
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- 아스키 코드 기준으로 소문자(a~z)가 대문자(A~Z)보다 뒤에 있음,문제 조건(대문자 < 소문자) 자동 만족된다.
3. **사용할 함수/라이브러리:**
   - `sorted()` : 문자열 정렬
   - `join()` : 리스트 → 문자열 변환

---
## ✅ 정답 코드 (Solution)

```python
def solution(s):
    answer = ''.join(sorted(s,reverse=True))
    return answer
```
