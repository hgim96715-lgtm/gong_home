## 문제 분석 & 설계 (Code Before Think)

> 알파벳으로 이루어진 문자열 `myString`과 `pat`이 주어집니다. `myString`의 연속된 부분 문자열 중 `pat`이 존재하면 1을 그렇지 않으면 0을 return 하는 solution 함수를 완성해 주세요.

단, 알파벳 대문자와 소문자는 구분하지 않습니다.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: myString문자열, pat문자열
    - **Output**:  `myString`의 연속된 부분 문자열 중 `pat`이 존재하면 `1` , 존재하지 않으면 `0`
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- 두 문자열을 모두 **소문자로 변환**
	- 문자열 포함 여부 연산자인 **`in`** 활용
3. **사용할 함수/라이브러리:**
   - `str.lower()` : 문자열을 소문자로 변환
   - `in` 연산자 : 부분 문자열 포함 여부 확인

---
## ✅ 정답 코드 (Solution)

```python
def solution(myString, pat):
    p_l=pat.lower()
    s_l=myString.lower()
    if p_l in s_l:
        return 1
    else:
        return 0

```

-  **새로 알게 된 함수/메서드:**

```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def solution(myString, pat):
    return int(pat.lower() in myString.lower())
```
