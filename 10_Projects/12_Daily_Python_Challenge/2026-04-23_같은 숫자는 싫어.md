## 문제 분석 & 설계 (Code Before Think)

> 배열 arr가 주어집니다. 배열 arr의 각 원소는 숫자 0부터 9까지로 이루어져 있습니다. 이때, 배열 arr에서 연속적으로 나타나는 숫자는 하나만 남기고 전부 제거하려고 합니다. 단, 제거된 후 남은 수들을 반환할 때는 배열 arr의 원소들의 순서를 유지해야 합니다. 예를 들면,

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 0~9 사이의 숫자로 이루어진 리스트 `arr`
    - **Output**: 연속된 중복만 제거된 리스트 (예: `[1, 3, 0, 1]`)
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)

3. **사용할 함수/라이브러리:**
   - `list.append()` (원소 추가)
   - `list[-1]` (리스트의 마지막 원소 접근)

---
## ✅ 정답 코드 (Solution)

```python
def solution(arr):
    answer = []
    for i in range(0,len(arr)):
        if len(answer)==0 or answer[-1]!=arr[i]:
            answer.append(arr[i])
    return answer
```

-  **내가 실수한 부분 (Syntax/Logic):**
    
-  **새로 알게 된 함수/메서드:**

	- **`answer[-1:]`**: 리스트 슬라이싱을 이용해 마지막 원소를 **리스트 형태**로 가져오는 방법. (빈 리스트일 때도 에러가 나지 않음)
