## 문제 분석 & 설계 (Code Before Think)

> 문자열 `my_str`과 `n`이 매개변수로 주어질 때, `my_str`을 길이 `n`씩 잘라서 저장한 배열을 return하도록 solution 함수를 완성해주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 문자열 `my_str`, 정수 `n` (자를 길이)
    - **Output**: 길이 `n`씩 잘린 문자열 리스트
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- 고정 개수가 아니라 “길이 기준 반복”
3. **사용할 함수/라이브러리:**
   - 

---
## ✅ 정답 코드 (Solution)

```python
def solution(my_str, n):
    answer=[]
    for i in range(0,len(my_str),n):
        answer.append(my_str[i:i+n])
    return answer
```

---
## 오답 노트 & 배운 점 (Retrospective)

-  **내가 실수한 부분 (Syntax/Logic):**
	- 처음에 `my_str[:i+n]` 으로 작성해서 항상 **0부터 누적 슬라이싱**이 되어버리는 현상이 발생했다. 
-  **새로 알게 된 함수/메서드:**
	- `[i:j]` = i부터 j까지 (**시작 위치 중요**)

## 더 나은 풀이 (Pythonic Code)

> 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?


```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def solution(my_str, n):
    return [my_str[i: i + n] for i in range(0, len(my_str), n)]
```
