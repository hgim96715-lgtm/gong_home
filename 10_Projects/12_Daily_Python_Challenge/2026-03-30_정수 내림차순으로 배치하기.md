## 문제 분석 & 설계 (Code Before Think)

> 함수 solution은 정수 n을 매개변수로 입력받습니다.
>  n의 각 자릿수를 큰것부터 작은 순으로 정렬한 새로운 정수를 리턴해주세요. 예를들어 n이 118372면 873211을 리턴하면 됩니다.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 정수 n
    - **Output**: 자릿수를 내림차순 정렬한 **정수**
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- 정수 → 문자열 → 정렬 → 다시 정수
3. **사용할 함수/라이브러리:**
   - `str(n)` → 정수를 문자열로 변환
   - `sorted(iterable, reverse=True)` → 내림차순 정렬
   - `"".join(list)` → 리스트를 문자열로 결합
   - `int()` → 문자열을 정수로 변환

---
## ✅ 정답 코드 (Solution)

```python
def solution(n):
    arr = [a for a in str(n)]
    arr2=sorted(arr,reverse=True)
    answer=''.join(arr2)
    return int(answer)
```

# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def solution(n):
    return int(''.join(sorted(str(n), reverse=True)))
