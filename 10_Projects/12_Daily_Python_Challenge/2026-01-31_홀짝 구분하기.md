## 문제 분석 & 설계 (Code Before Think)

> 자연수 `n`이 입력으로 주어졌을 때 
> 만약 `n`이 짝수이면 "`n` is even"을, 홀수이면 "`n` is odd"를 출력하는 코드를 작성해 보세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
	- **입력:** 자연수 `n`
	- **출력:**
	    - `n`이 짝수이면 `"n is even"`
	    - `n`이 홀수이면 `"n is odd"`
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- `n % 2 == 0` → 짝수
	- 그렇지 않으면 → 홀수
3. **사용할 함수/라이브러리:**
   - `if-else` **삼항 연산자** : 조건에 따라 결과 문자열 결정

---
## ✅ 정답 코드 (Solution)

```python
n = int(input())
result = f"{n} is even" if n % 2 == 0 else f"{n} is odd"
print(result)
```

## 더 나은 풀이 (Pythonic Code)

> 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?


```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
N = int(input())
print(f"{N} is {'even' if N % 2 == 0 else 'odd'}")
```

- "N is"는 중복이니깐, **`even` / `odd`만 출력**하도록 하는 방식이 더 깔끔했겠다.
