## 문제 분석 & 설계 (Code Before Think)

> 문자열 `my_string`과 이차원 정수 배열 `queries`가 매개변수로 주어집니다. 
> `queries`의 원소는 [s, e] 형태로, `my_string`의 인덱스 s부터 인덱스 e까지를 뒤집으라는 의미입니다. `my_string`에 `queries`의 명령을 순서대로 처리한 후의 문자열을 return 하는 solution 함수를 작성해 주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: `my_string` : 문자열
    - **Output**: 모든 `queries`를 순서대로 적용한 **최종 문자열**
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- `queries`를 **순차적으로 반복**하며 `my_string`을 계속 업데이트
3. **사용할 함수/라이브러리:**
   -  슬라이싱 step 활용 (`[:​:-1]`)
   - 기본 반복문 (`for`)

---
## ✅ 정답 코드 (Solution)

```python
def solution(my_string, queries):
    for q in queries:
        a,b=q
        my_string=my_string[:a]+my_string[a:b+1][::-1]+my_string[b+1:]
    return my_string
```
