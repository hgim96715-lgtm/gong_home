## 문제 분석 & 설계 (Code Before Think)

> 임의의 양의 정수 n에 대해, n이 어떤 양의 정수 x의 제곱인지 아닌지 판단하려 합니다.  
> n이 양의 정수 x의 제곱이라면 x+1의 제곱을 리턴하고, n이 양의 정수 x의 제곱이 아니라면 -1을 리턴하는 함수를 완성하세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 양의 정수 `n`
    - **Output**: `n`이 어떤 정수 `x`의 제곱이면 `(x+1)**2` 반환 아니면 `-1` 반환
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)

3. **사용할 함수/라이브러리:**
   - `math.sqrt(n)` 제곱근 계산
   - `int()`실수를 정수로 변환 (버림)
---
## ✅ 정답 코드 (Solution)

```python
import math
def solution(n):
    answer = (math.sqrt(n)+1)**2 if int(math.sqrt(n))**2==n else -1
    return answer
```

## 오답 노트 & 배운 점 (Retrospective)

-  **내가 실수한 부분 (Syntax/Logic):**
    - `math.sqrt(n)`의 결과는 **실수(float)** 인데,  그대로 비교해서 오차가 발생할 수 있음
    - `int()`를 사용하지 않아 정확한 제곱수 판별이 안 될 수 있어서 테스트에서 2개의 실패가 나왔다.
-  **새로 알게 된 함수/메서드:**
	- "판별 문제는 정수 연산으로 바꿔서 해결한다"
