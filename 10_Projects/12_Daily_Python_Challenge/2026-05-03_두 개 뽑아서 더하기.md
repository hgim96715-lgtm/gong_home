## 문제 분석 & 설계 (Code Before Think)

> 정수 배열 numbers가 주어집니다. numbers에서 서로 다른 인덱스에 있는 두 개의 수를 뽑아 더해서 만들 수 있는 모든 수를 배열에 오름차순으로 담아 return 하도록 solution 함수를 완성해주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 정수 배열 `numbers`
    - **Output**: 서로 다른 인덱스의 두 수를 더해서 만들 수 있는 모든 합을 중복 없이 오름차순으로 정렬한 배열
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- `i`는 첫 번째 숫자의 인덱스, `j`는 두 번째 숫자의 인덱스로 사용
	- 같은 숫자 조합을 중복해서 확인하지 않기 위해 `j`는 `i + 1`부터 시작
3. **사용할 함수/라이브러리:**
    - `range()`
    - `len()`
    - `sorted()`


---
## ✅ 정답 코드 (Solution)

```python
def solution(numbers):
    answer = []
    for i in range(len(numbers)):
        for j in range(i + 1, len(numbers)):
            if numbers[i]+numbers[j] in answer:
                continue
            else:
                answer.append(numbers[i]+numbers[j])
    return sorted(answer)
```

---
## 오답 노트 & 배운 점 (Retrospective)

-  **내가 실수한 부분 (Syntax/Logic):**
    - 처음에는 `for i in numbers`, `for j in numbers`처럼 값 기준으로 반복하려고 했다
    - 하지만 이 방식은 같은 인덱스의 숫자끼리 더하는 경우도 생길 수 있다
-  **새로 알게 된 함수/메서드:**
	- `range(len(numbers))`를 사용하면 배열의 인덱스를 기준으로 반복
	- `j`를 `i + 1`부터 시작하면 같은 조합을 중복해서 계산하지 않을 수 있다.

```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def solution(numbers):
    answer = set()

    for i in range(len(numbers)):
        for j in range(i + 1, len(numbers)):
            answer.add(numbers[i] + numbers[j])

    return sorted(answer)
```

>더 간단하게는 일단 모든 합을 `answer`에 넣고, 마지막에 `set()`으로 중복을 제거할 수 있다.
