## 문제 분석 & 설계 (Code Before Think)

> 프로그래머스 모바일은 개인정보 보호를 위해 고지서를 보낼 때 고객들의 전화번호의 일부를 가립니다.  
> 전화번호가 문자열 phone_number로 주어졌을 때, 전화번호의 뒷 4자리를 제외한 나머지 숫자를 전부 `*`으로 가린 문자열을 리턴하는 함수, solution을 완성해주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 문자열 `phone_number` (숫자로 이루어진 전화번호)
    - **Output**: 뒷 4자리를 제외한 나머지를 `*`로 가린 문자열
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- 전체 길이에서 4를 뺀 만큼 `*`을 생성한다
	- `*`로 만든 문자열과 뒤 4자리를 이어 붙인다
3. **사용할 함수/라이브러리:**
   - 문자열 슬라이싱 `[-4:]` : 뒤 4자리 추출
   - 문자열 곱셈 `"*" * n` : 반복 문자열 생성

---
## ✅ 정답 코드 (Solution)

```python
def solution(phone_number):
    answer = (len(phone_number)-4)*'*'+phone_number[-4:]
    return answer
```
