## 문제 분석 & 설계 (Code Before Think)

> 대문자와 소문자가 섞여있는 문자열 s가 주어집니다. s에 'p'의 개수와 'y'의 개수를 비교해 같으면 True, 다르면 False를 return 하는 solution를 완성하세요. 'p', 'y' 모두 하나도 없는 경우는 항상 True를 리턴합니다. 단, 개수를 비교할 때 대문자와 소문자는 구별하지 않습니다.
> 예를 들어 s가 "pPoooyY"면 true를 return하고 "Pyy"라면 false를 return합니다.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 문자열 `s` (대문자 + 소문자 혼합 가능)
    - **Output**: `bool` (True 또는 False)
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- 문자열을 **대소문자 구분 없이 비교해야 하므로 → 소문자로 변환**
	- `'p'`와 `'y'`의 개수를 각각 센다
	- 두 개수가 같으면 `True`, 다르면 `False`
3. **사용할 함수/라이브러리:**
   - `s.lower()`로 소문자 변환
   - `collections.Counter` → 문자 개수 자동 카운팅

---
## ✅ 정답 코드 (Solution)

```python
from collections import Counter
def solution(s):
    ls=s.lower()
    c_s=Counter(ls)
    return True if c_s['p']==c_s['y'] else False
```
