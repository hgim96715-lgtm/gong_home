## ✅ 정답 코드 (Solution)

```python
def solution(lines):
    count = [0] * 201 
    
    for start, end in lines:
        for i in range(start, end):
            count[i + 100] += 1  
    
    answer = 0
    for c in count:
        if c >= 2:
            answer += 1
    
    return answer
```

## 더 나은 풀이 (Pythonic Code)

> 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?


```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def solution(lines):
    s1 = set(i for i in range(lines[0][0], lines[0][1]))
    s2 = set(i for i in range(lines[1][0], lines[1][1]))
    s3 = set(i for i in range(lines[2][0], lines[2][1]))
    return len((s1 & s2) | (s2 & s3) | (s1 & s3))
```
