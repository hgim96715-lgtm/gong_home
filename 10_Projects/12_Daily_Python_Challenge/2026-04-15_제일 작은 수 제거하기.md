## 문제 분석 & 설계 (Code Before Think)

> 정수를 저장한 배열, arr 에서 가장 작은 수를 제거한 배열을 리턴하는 함수, solution을 완성해주세요. 
> 단, 리턴하려는 배열이 빈 배열인 경우엔 배열에 -1을 채워 리턴하세요.
>  예를들어 arr이 [4,3,2,1]인 경우는 [4,3,2]를 리턴 하고, [10]면 [-1]을 리턴 합니다.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 정수 리스트 `arr`
    - **Output**: 가장 작은 값을 제거한 리스트 단, 결과가 빈 리스트면 `[-1]` 반환
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- 리스트에서 **최솟값을 찾는다** → `min(arr)`
	- 해당 값을 리스트에서 제거 → `remove()`
	- 리스트가 비어있으면 → `[-1]`
3. **사용할 함수/라이브러리:**
   - `min()` : 최솟값 찾기
   - `list.remove()` : 특정 값 제거

---
## ✅ 정답 코드 (Solution)

```python
def solution(arr):
    arr.remove(min(arr))
    return [-1] if len(arr)==0 else arr
```
