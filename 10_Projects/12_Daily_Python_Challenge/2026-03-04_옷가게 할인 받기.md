## 문제 분석 & 설계 (Code Before Think)

> 머쓱이네 옷가게는 10만 원 이상 사면 5%, 30만 원 이상 사면 10%, 50만 원 이상 사면 20%를 할인해줍니다.  
> 구매한 옷의 가격 `price`가 주어질 때, 지불해야 할 금액을 return 하도록 solution 함수를 완성해보세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 정수형 변수 `price`
    - **Output**: 할인 정책이 적용된 후 **실제로 지불해야 할 금액**
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- **조건 분기(if / elif / else)** 를 사용해 가격 구간별 할인율 적용
	- 최종 결과는 **정수(int)** 로 반환
3. **사용할 함수/라이브러리:**
   - `if / elif / else` : 가격 구간별 조건 처리

## ✅ 정답 코드 (Solution)

```python
def solution(price):
    answer = 0
    if price>=500000:
        answer=price-(price*0.2)
    elif price>=300000:
        answer=price-(price*0.1)
    elif price>=100000:
        answer=price-(price*0.05)
    else:
        answer=price
    return int(answer)
```

## 더 나은 풀이 (Pythonic Code)

> 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?


```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def solution(price):
    discount_rates = {500000: 0.8, 300000: 0.9, 100000: 0.95, 0: 1}
    for discount_price, discount_rate in discount_rates.items():
        if price >= discount_price:
            return int(price * discount_rate)
```
