## 문제 분석 & 설계 (Code Before Think)

> `my_string`은 "3 + 5"처럼 문자열로 된 수식입니다. 문자열 `my_string`이 매개변수로 주어질 때, 수식을 계산한 값을 return 하는 solution 함수를 완성해주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 공백으로 구분된 문자열 수식
    - **Output**: 수식을 계산한 **정수 결과값**
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- `split()`으로 나누어 **숫자와 연산자 리스트**로 변환
3. **사용할 함수/라이브러리:**
   - `split()`
   - `enumerate()`

---
## ✅ 정답 코드 (Solution)

```python
def solution(my_string):
    my_string = my_string.split()
    answer = int(my_string[0])
    
    for i, s in enumerate(my_string):
        if s == '+':
            answer += int(my_string[i+1])
        elif s == '-':
            answer -= int(my_string[i+1])
            
    return answer
```

```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def solution(my_string):
    return sum(int(i) for i in my_string.replace(' - ', ' + -').split(' + '))
```
