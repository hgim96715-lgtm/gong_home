## 문제 분석 & 설계 (Code Before Think)

> 어떤 정수들이 있습니다. 이 정수들의 절댓값을 차례대로 담은 정수 배열 absolutes와 이 정수들의 부호를 차례대로 담은 불리언 배열 signs가 매개변수로 주어집니다. 실제 정수들의 합을 구하여 return 하도록 solution 함수를 완성해주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 
	    - `absolutes` → 정수의 절댓값이 담긴 리스트 (List[int])
	    - `signs` → 각 숫자의 부호를 나타내는 불리언 리스트 (List[bool])
	    - `True` = 양수 / `False` = 음수
    - **Output**: 실제 정수들의 합 (int)
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- `absolutes`와 `signs`를 **같은 인덱스 기준으로 묶기 (zip 사용)**
3. **사용할 함수/라이브러리:**
   - `zip()` → 두 리스트를 묶어서 동시에 순회

---
## ✅ 정답 코드 (Solution)

```python
def solution(absolutes, signs):
    answer = 0
    for a,s in zip(absolutes,signs):
        if s:
            answer+=a
        else:
            answer-=a
    return answer
```

```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def solution(absolutes, signs):
    return sum(absolutes if sign else -absolutes for absolutes, sign in zip(absolutes, signs))
```
