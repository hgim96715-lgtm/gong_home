## 문제 분석 & 설계 (Code Before Think)

> 정수 `n`이 매개변수로 주어질 때, `n`의 약수를 오름차순으로 담은 배열을 return하도록 solution 함수를 완성해주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 정수 n 
    - **Output**: `n`의 약수를 **오름차순으로 담은 리스트**
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- 반복문에서 현재 숫자 `i`가 `n`을 **나누어 떨어지면 (`n % i == 0`)** 
	- `i`는 `n`의 약수이므로 `answer` 리스트에 추가한다.
3. **사용할 함수/라이브러리:**
   - `range()` : 반복문 범위 생성
   - `%` : 나머지 연산

---
## ✅ 정답 코드 (Solution)

```python
def solution(n):
    answer = []
    for i in range(1,n+1):
        if n%i==0:
            answer.append(i)
    return answer
```

## 더 나은 풀이 (Pythonic Code)

> 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?


```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def solution(n):
    answer = [i for i in range(1,n+1) if n%i == 0]
    return answer
```
