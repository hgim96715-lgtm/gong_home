## 문제 분석 & 설계 (Code Before Think)

> 배열 array의 i번째 숫자부터 j번째 숫자까지 자르고 정렬했을 때, k번째에 있는 수를 구하려 합니다.
> 예를 들어 array가 [1, 5, 2, 6, 3, 7, 4], i = 2, j = 5, k = 3이라면
> . array의 2번째부터 5번째까지 자르면 [5, 2, 6, 3]입니다
>  1에서 나온 배열을 정렬하면 [2, 3, 5, 6]입니다.
>   2에서 나온 배열의 3번째 숫자는 5입니다.

2. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: `array`: 원본 숫자 배열 `commands`:  2차원 배열
    - **Output**: 각 명령마다 `array`를 `i`번째부터 `j`번째까지 자르고 정렬한 뒤, `k`번째 수를 뽑아 담은 배열
3. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- `commands`를 하나씩 순회
	- 각 command에서 `i, j, k`를 꺼낸다.
	- 자른 배열을 `sorted()`로 정렬한다.
4. **사용할 함수/라이브러리:**
    - 리스트 슬라이싱 `array[i-1:j]`
    - `sorted()`
    - `append()`

---
## ✅ 정답 코드 (Solution)

```python
def solution(array, commands):
    answer = []
    for command in commands:
        i,j,k=command
        array_cp=sorted(array[i-1:j])
        answer.append(array_cp[k-1])
    return answer
```

## 더 나은 풀이 (Pythonic Code)

> 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
> `map()`과 `lambda`를 사용하면 한 줄로도 해결할 수 있다


```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def solution(array, commands):
    return list(map(lambda x:sorted(array[x[0]-1:x[1]])[x[2]-1], commands))
```
