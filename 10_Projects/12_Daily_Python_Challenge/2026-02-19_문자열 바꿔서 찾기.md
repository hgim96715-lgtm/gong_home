## 문제 분석 & 설계 (Code Before Think)

> 문자 "A"와 "B"로 이루어진 문자열 `myString`과 `pat`가 주어집니다.
>  `myString`의 "A"를 "B"로, "B"를 "A"로 바꾼 문자열의 연속하는 부분 문자열 중 `pat`이 있으면 1을 아니면 0을 return 하는 solution 함수를 완성하세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 문자열 `myString` , `pat`
    - **Output**: 부분 문자열 중 `pat`이 있으면 1을 아니면 0
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)

3. **사용할 함수/라이브러리:**
   - `translate`

---
## ✅ 정답 코드 (Solution)

```python
def solution(myString, pat):
    result = myString.translate(str.maketrans("AB", "BA"))
    answer = 1 if pat in result else 0
    return answer
```
