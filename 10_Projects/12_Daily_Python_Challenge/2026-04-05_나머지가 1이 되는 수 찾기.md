## 문제 분석 & 설계 (Code Before Think)

> 자연수 `n`이 매개변수로 주어집니다. 
> `n`을 `x`로 나눈 나머지가 1이 되도록 하는 가장 작은 자연수 `x`를 return 하도록 solution 함수를 완성해주세요. 답이 항상 존재함은 증명될 수 있습니다.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 자연수 `n`
    - **Output**: `n % x == 1`을 만족하는 **가장 작은 자연수 `x`**
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- `x`는 1부터 `n`까지 탐색 가능
3. **사용할 함수/라이브러리:**
   - `%` (나머지 연산)
   - `for` 반복문
   - `min()` 최소값
---
## ✅ 정답 코드 (Solution)

```python
def solution(n):
    answer = []
    for k in range(1,n+1):
        if n%k==1:
            answer.append(k)
    return min(answer)
```

## 더 나은 풀이 (Pythonic Code)

> 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?


```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def solution(n):
    return next(x for x in range(1, n+1) if n % x == 1)
```
