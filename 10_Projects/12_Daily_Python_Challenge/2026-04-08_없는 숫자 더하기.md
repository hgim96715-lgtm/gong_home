## 문제 분석 & 설계 (Code Before Think)

> 0부터 9까지의 숫자 중 일부가 들어있는 정수 배열 `numbers`가 매개변수로 주어집니다. 
> `numbers`에서 찾을 수 없는 0부터 9까지의 숫자를 모두 찾아 더한 수를 return 하도록 solution 함수를 완성해주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 정수 배열 `numbers` (0~9 사이 숫자, 일부 누락됨)
    - **Output**: `numbers`에 없는 숫자들의 합 (정수)
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- 각 숫자가 `numbers`에 포함되어 있는지 확인
	- 포함되어 있지 않으면 결과값에 더함
3. **사용할 함수/라이브러리:**
   - `range()` : 0~9 순회
   - `sum()` : 리스트 합 계산

---
## ✅ 정답 코드 (Solution)

```python
def solution(numbers):
    answer = 0
    for n in range(0,10):
        if n not in numbers:
            answer+=n
    return answer
```

	- `sum()`을 활용하면 반복문 없이도 합을 쉽게 구할 수 있음
	- 0~9의 합은 항상 **45**
	- 없는 숫자의 합 = 45 - 현재 숫자들의 합
