## 문제 분석 & 설계 (Code Before Think)

> 첫 번째 분수의 분자와 분모를 뜻하는 `numer1`, `denom1`, 두 번째 분수의 분자와 분모를 뜻하는 `numer2`, `denom2`가 매개변수로 주어집니다. 
> 두 분수를 더한 값을 기약 분수로 나타냈을 때 분자와 분모를 순서대로 담은 배열을 return 하도록 solution 함수를 완성해보세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: `numer1`, `denom1`, `numer2`, `denom2`
    - **Output**: 분자와 분모를 순서대로 담은 배열
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- 일된 분모에 맞게 **분자를 계산** >`numer1 * denom2 + numer2 * denom1`
	- “덧셈을 먼저 하고, **그 결과를 기준으로** 약분한다”
3. **사용할 함수/라이브러리:**
   - `math.gcd(a, b)`
   - `math`

## ✅ 정답 코드 (Solution)

```python
import math
def solution(numer1, denom1, numer2, denom2):
    one=(numer1*denom2+numer2*denom1, denom1*denom2)
    gcd=math.gcd(numer1*denom2+numer2*denom1,denom1*denom2)
    return [(numer1*denom2+numer2*denom1)/gcd,denom1*denom2/gcd]
```

## 오답 노트 & 배운 점 (Retrospective)

-  **내가 실수한 부분 (Syntax/Logic):**
	- 처음에 `gcd(denom1, denom2)`를 사용했지만, **약분은 덧셈이 끝난 후의 분자와 분모 기준**으로 해야 한다는 점을 놓쳤다.
    
-  **새로 알게 된 함수/메서드:**
	- `fractions.Fraction` 
	- 파이썬 표준 라이브러리에서 제공하는 **분수 전용 클래스**
	- 분수의 덧셈, 약분을 **자동으로 처리**해 준다.
