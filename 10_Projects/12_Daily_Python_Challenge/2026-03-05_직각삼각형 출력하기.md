## 문제 분석 & 설계 (Code Before Think)

> "`*`"의 높이와 너비를 1이라고 했을 때, "`*`"을 이용해 직각 이등변 삼각형을 그리려고합니다. 정수 n 이 주어지면 높이와 너비가 n 인 직각 이등변 삼각형을 출력하도록 코드를 작성해보세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 정수 n
    - **Output**: 이등변 삼각형을 출력
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)

3. **사용할 함수/라이브러리:**
   - `range()`
   - `str.join()`

---
## ✅ 정답 코드 (Solution)

```python
n = int(input())

answer=[i*'*' for i in range(1,n+1)]

print('\n'.join(answer))
```

## 더 나은 풀이 (Pythonic Code)

> 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?


```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
n = int(input())

print('\n'.join('*' * i for i in range(1, n + 1)))
```
