## 문제 분석 & 설계 (Code Before Think)

> 머쓱이네 피자가게는 피자를 여섯 조각으로 잘라 줍니다. 피자를 나눠먹을 사람의 수 `n`이 매개변수로 주어질 때, `n`명이 주문한 피자를 남기지 않고 모두 같은 수의 피자 조각을 먹어야 한다면 최소 몇 판을 시켜야 하는지를 return 하도록 solution 함수를 완성해보세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 피자를 나눠 먹을 사람 수 `n` (정수)
    - **Output**: 모든 사람이 같은 수의 피자 조각을 먹기 위해 필요한 **최소 피자 판 수** (정수)
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- `n × 6 ÷ gcd(n, 6)`
3. **사용할 함수/라이브러리:**
   - `math.gcd()` : 두 수의 최대공약수 계산

---
## ✅ 정답 코드 (Solution)

```python
import math
def solution(n):
    gcd = math.gcd(n,6)
    answer=n/gcd
    return answer
```

## 더 나은 풀이 (Pythonic Code)

> 문제에서 요구하는 값은 **정수 판 수**이기 때문에 실수 나눗셈(`/`) 대신 **정수 나눗셈(`//`)** 을 사용하는 것이 더 명확할것 같다.


```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
import math

def solution(n):
    return n // math.gcd(n, 6)
```
