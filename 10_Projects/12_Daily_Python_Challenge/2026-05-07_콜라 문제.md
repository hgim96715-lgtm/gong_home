## 문제 분석 & 설계 (Code Before Think)

> 콜라 빈 병 2개를 가져다주면 콜라 1병을 주는 마트가 있다. 빈 병 20개를 가져다주면 몇 병을 받을 수 있는가?
> 단, 보유 중인 빈 병이 2개 미만이면, 콜라를 받을 수 없다.
>콜라를 받기 위해 마트에 주어야 하는 병 수 `a`, 빈 병 a개를 가져다 주면 마트가 주는 콜라 병 수 `b`, 상빈이가 가지고 있는 빈 병의 개수 `n`이 매개변수로 주어집니다. 상빈이가 받을 수 있는 콜라의 병 수를 return 하도록 solution 함수를 작성해주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: `a`: 콜라를 받기 위해 마트에 줘야 하는 빈 병의 개수, `b`: 빈 병 `a`개를 주면 받을 수 있는 콜라 병 수, `n`: 처음 가지고 있는 빈 병의 개수
    - **Output**: 상빈이가 받을 수 있는 콜라의 총 병 수
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- 현재 가지고 있는 빈 병의 개수 `n`이 `a`개 이상이면 계속 교환할 수 있다.
	- 교환 횟수는 `n // a`
	- 새로 받는 콜라 수는 `(n // a) * b`
	- 교환하고 남은 빈 병은 `n % a`
3. **사용할 함수/라이브러리:**
	- `while` 반복문

---
## ✅ 정답 코드 (Solution)

```python
def solution(a, b, n):
    answer = 0
    while n>=a:
        first=n//a
        answer+=first*b
        n=(n%a)+(first*b)
        # print(n,n%a,first*b)
    return answer
```

## 더 나은 풀이 (Pythonic Code)

> 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?

>빈 병 `a`개를 주고 콜라 `b`병을 받으면, 받은 콜라는 다시 마신 뒤 빈 병
>교환 후 남은 빈 병을 버리지 않고 다음 계산에 포함

```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
solution = lambda a, b, n: max(n - b, 0) // (a - b) * b
```
