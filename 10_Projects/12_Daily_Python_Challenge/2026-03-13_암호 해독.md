## 더 나은 풀이 (Pythonic Code)

> 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?


```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def solution(cipher, code):
    answer = cipher[code-1::code]
    return answer
```

>더 깔끔하다.
>`::code` → code 간격으로 글자 선택
>반복문 없이 한 줄로 결과 반환 가능
