> 양의 정수 `n`이 매개변수로 주어집니다. ${n} x {n}$배열에 1부터 ${n}^2$까지 정수를 인덱스 $[0][0]$부터 시계방향 나선형으로 배치한 이차원 배열을 return 하는 solution 함수를 작성해 주세요.

## ✅ 정답 코드 (Solution)

```python
def solution(n):
    arr = [[0] * n for _ in range(n)]
    
    # 오른쪽, 아래, 왼쪽, 위
    dx = [0, 1, 0, -1]
    dy = [1, 0, -1, 0]
    
    x, y, dir = 0, 0, 0
    
    for num in range(1, n*n + 1):
        arr[x][y] = num
        
        nx, ny = x + dx[dir], y + dy[dir]
        
        # 범위 벗어나거나 이미 채워진 곳이면 방향 전환
        if nx < 0 or nx >= n or ny < 0 or ny >= n or arr[nx][ny] != 0:
            dir = (dir + 1) % 4
            nx, ny = x + dx[dir], y + dy[dir]
        
        x, y = nx, ny
    
    return arr
```

# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def solution(n):
    answer = [([0]*n) for _ in range(n)]
    j = 1
    top, bottom, right, left = 0, n-1, n-1, 0

    while j <= n**2:
        # 오른쪽
        for i in range(left, right+1):
            answer[top][i] = j
            j += 1
        top += 1

        # 아래쪽
        for i in range(top, bottom+1):
            answer[i][right] = j
            j += 1
        right -= 1

        # 왼쪽
        for i in range(right,left-1,-1):
            answer[bottom][i] = j
            j += 1
        bottom -= 1

        # 위쪽 
        for i in range(bottom, top-1,-1):
            answer[i][left] = j
            j += 1
        left += 1

    return answer
