## 문제 분석 & 설계 (Code Before Think)

> 함수 solution은 정수 x와 자연수 n을 입력 받아, x부터 시작해 x씩 증가하는 숫자를 n개 지니는 리스트를 리턴해야 합니다. 
> 다음 제한 조건을 보고, 조건을 만족하는 함수, solution을 완성해주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: `x`: 정수 (양수/음수 가능) , `n`: 자연수 (1 이상)
    - **Output**: 길이가 `n`인 리스트
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- n번 반복
3. **사용할 함수/라이브러리:**
   - `while` 반복문 → n개 만들기
   - `append()` → 값 추가

---
## ✅ 정답 코드 (Solution)

```python
def solution(x, n):
    answer = []
    a=x
    while len(answer)<n:
        answer.append(a)
        a+=x
    return answer
```

## 더 나은 풀이 (Pythonic Code)

> 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?


```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def solution(x, n):
    return [x * i for i in range(1, n + 1)]
```

>x, 2x, 3x, ... nx  , 이 문제는 "누적"이 아니라 "배수" 문제 라고 생각했다면 더 쉽게 풀기 좋았을 문제인것 같다.
