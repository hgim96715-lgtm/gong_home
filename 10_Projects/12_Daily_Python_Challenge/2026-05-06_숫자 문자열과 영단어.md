## 문제 분석 & 설계 (Code Before Think)

>  숫자의 일부 자릿수가 영단어로 바뀌어졌거나, 혹은 바뀌지 않고 그대로인 문자열 `s`가 매개변수로 주어집니다. `s`가 의미하는 원래 숫자를 return 하도록 solution 함수를 완성해주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 숫자와 영단어가 섞여 있는 문자열 `s`
    - **Output**: 문자열 `s`가 의미하는 원래 숫자
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- 숫자 영단어와 실제 숫자를 매핑한 딕셔너리를 만든다.
	- 딕셔너리를 순회하면서 문자열 `s` 안에 있는 영단어를 숫자 문자로 치환
	- `int()`로 정수로 변환해서 반환
3. **사용할 함수/라이브러리:**
	- `replace()`
	- `int()`

---
## ✅ 정답 코드 (Solution)

```python
def solution(s):
    en_word={"zero":"0","one":"1","two":"2","three":"3","four":"4","five":"5","six":"6","seven":"7","eight":"8","nine":"9"}
    for k,v in en_word.items():
         s = s.replace(k, v)
    return int(s)
```

## 오답 노트 & 배운 점 (Retrospective)

-  **내가 실수한 부분 (Syntax/Logic):**
    -  처음에는 `if k in s or v in s:`로 문자열 안에 해당 영단어나 숫자가 있는지만 확인했다.
    - 하지만 이 방식은 문자열의 실제 순서를 보존하지 못한다.
-  **새로 알게 된 함수/메서드:**
	- `replace(old, new)`
	- 문자열에서 `old`에 해당하는 값을 모두 찾아 `new`로 바꿔준다.

## 더 나은 풀이 (Pythonic Code)

> 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?


```python
# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
def solution(s):
    numbers = ["zero", "one", "two", "three", "four",
               "five", "six", "seven", "eight", "nine"]

    for i, word in enumerate(numbers):
        s = s.replace(word, str(i))

    return int(s)
```
