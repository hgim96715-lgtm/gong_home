## 문제 분석 & 설계 (Code Before Think)

> 이 문제에는 표준 입력으로 두 개의 정수 n과 m이 주어집니다.  
> 별(`*`) 문자를 이용해 가로의 길이가 n, 세로의 길이가 m인 직사각형 형태를 출력해보세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 공백으로 구분된 두 정수 `n`(가로 길이), `m`(세로 길이)
    - **Output**: `*`로 이루어진 가로 `n`, 세로 `m` 크기의 직사각형 문자열 출력
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- 한 줄에 `*`을 `n`개 생성
	- 해당 줄을 `m`번 반복하여 출력
3. **사용할 함수/라이브러리:**
	- `map()`: 문자열 → 정수 변환
	- 문자열 곱셈 (`*`): 반복 문자열 생성

---
## ✅ 정답 코드 (Solution)

```python
a, b = map(int, input().strip().split(' '))
c=('*'*a+'\n')*b
print(c)
```
