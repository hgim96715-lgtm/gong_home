## 문제 분석 & 설계 (Code Before Think)

> 정수 `num1`, `num2`가 매개변수로 주어질 때, `num1`을 `num2`로 나눈 몫을 return 하도록 solution 함수를 완성해주세요.

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: 정수 `num1`, `num2`
    - **Output**: `num1 ÷ num2`의 **정수 몫**
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)

3. **사용할 함수/라이브러리:**
   - `//` : 정수 나눗셈 (floor division)

## ✅ 정답 코드 (Solution)

```python
def solution(num1, num2):
    return num1//num2
```
