## 문제 분석 & 설계 (Code Before Think)

> 2차원 좌표 평면에 변이 축과 평행한 직사각형이 있습니다. 
> 직사각형 네 꼭짓점의 좌표 [[x1, y1], [x2, y2], [x3, y3], [x4, y4]]가 담겨있는 배열 `dots`가 매개변수로 주어질 때, 직사각형의 넓이를 return 하도록 solution 함수를 완성해보세요

1. **입/출력 분석:** (무엇을 받고, 무엇을 돌려주는가?)
    - **Input**: `dots` → 4개의 좌표가 담긴 리스트 `[[x1, y1], ..., [x4, y4]]`
    - **Output**: 직사각형의 넓이 (정수)
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)
	- 최댓값과 최솟값의 차이 → 가로 길이, y 최댓값과 최솟값의 차이 → 세로 길이
3. **사용할 함수/라이브러리:**
   - `max()` : 최대값 구하기
   - `min()` : 최소값 구하기

---
## ✅ 정답 코드 (Solution)

```python
def solution(dots):
    answer = 0
    a,b,c,d=dots
    e=max(a[0],b[0],c[0],d[0])-min(a[0],b[0],c[0],d[0])
    f=max(a[1],b[1],c[1],d[1])-min(a[1],b[1],c[1],d[1])
    answer=e*f
    return answer
```

# 다른 사람의 풀이나, 더 간결한 코드(리팩토링)가 있다면?
