## 문제 분석 & 설계 (Code Before Think)

> 문자열 리스트 `str_list`에는 "u", "d", "l", "r" 네 개의 문자열이 여러 개 저장되어 있습니다.
>  `str_list`에서 "l"과 "r" 중 먼저 나오는 문자열이 "l"이라면 해당 문자열을 기준으로 왼쪽에 있는 문자열들을 순서대로 담은 리스트를, 먼저 나오는 문자열이 "r"이라면 해당 문자열을 기준으로 오른쪽에 있는 문자열들을 순서대로 담은 리스트를 return하도록 solution 함수를 완성해주세요.
>   "l"이나 "r"이 없다면 빈 리스트를 return합니다.

    - **Input**:  문자열 리스트 `str_list`
    - **Output**: 
2. **핵심 로직:** (어떤 자료구조나 흐름을 쓸 것인가?)

3. **사용할 함수/라이브러리:**
   - `in` : 리스트에 값이 존재하는지 True/False 반환
   -  `index()` : 해당 값의 첫 번째 인덱스 반환 (없으면 ValueError)

## ✅ 정답 코드 (Solution)

```python
def solution(str_list):
    answer = []
    l_in="l" in str_list
    r_in="r" in str_list
    if l_in and r_in:
        l_idx = str_list.index("l") 
        r_idx=str_list.index("r")
        if l_idx<r_idx:
            answer=str_list[:l_idx]
        else:
            answer=str_list[r_idx+1:]
    elif l_in:
        l_idx = str_list.index("l") 
        answer=str_list[:l_idx]
    elif r_in:
        r_idx=str_list.index("r")
        answer=str_list[r_idx+1:]
    return answer
```

## 오답 노트 & 배운 점 (Retrospective)

-  **내가 실수한 부분 (Syntax/Logic):**
	- `index()` 사용 전 존재 여부를 확인하지 않으면 ValueError 발생 
    
-  **새로 알게 된 함수/메서드:**
	- `list.index(value)` : 값의 첫 번째 위치 반환, 없으면 에러 → 반드시 `in`으로 존재 확인 후 사용
