### 🔷 섹션 1 — pwd 와 cd 로 홈 디렉터리 확인하기

**개념 요약**

- **pwd (print working directory):** 현재 내가 작업 중인 디렉터리의 전체 경로를 출력합니다. 길을 잃었을 때 위치를 확인하는 용도입니다.
- **cd (change directory):** 다른 디렉터리로 이동할 때 사용합니다.

**명령어 / 문법**

```bash
pwd        # 현재 위치 확인 (예: /home/labex/project)
cd         # 인자 없이 입력 시 홈 디렉터리로 즉시 이동
pwd        # 이동 후 확인 (예: /home/labex)
cd ~/project  # 홈 디렉터리 하위의 project 폴더로 이동
```

**실습 메모**

- 터미널을 처음 열었을 때나 작업 중에 내 위치가 헷갈린다면 `pwd`부터 입력하자.

### 🔷 섹션 2 — 절대 경로와 물결표 확장을 사용하여 디렉터리 변경하기

**개념 요약**

- **절대 경로:** 최상위 루트(`/`)부터 시작하는 고유한 경로입니다. 어디에 있든 정확한 위치로 이동할 수 있습니다.
- **물결표 (~):** 현재 로그인한 사용자의 홈 디렉터리(`/home/사용자명`)를 가리키는 단축키입니다.

**명령어 / 문법**

```bash
cd /etc    # 절대 경로를 이용해 설정 디렉터리로 이동
cd ~       # 어디서든 홈 디렉터리로 복귀
cd ~/project # 홈 디렉터리 기준의 경로로 이동
```

**실습 메모**

- `/`는 뿌리(root), `~`는 내 방(home)이라고 생각하면 쉽다.

### 🔷 섹션 3 — ls 와 ls -a 로 디렉터리 내용 나열하기

**개념 요약**

- **touch:** 내용이 없는 빈 파일을 생성하거나 파일의 수정 시간을 갱신합니다.
- **ls:** 현재 디렉터리의 파일 목록을 보여줍니다.
- **ls -a (--all):** 이름이 점(`.`)으로 시작하는 숨겨진 파일(hidden files)까지 모두 표시합니다.

**명령어 / 문법**

```bash
touch sample.txt touch .hidden_file  # 일반 파일과 숨김 파일 생성
ls # sample.txt
ls -a # . .. .hidden_file sample.txt
```

**실습 메모**

- 리눅스에서 `.`으로 시작하는 파일은 설정 파일인 경우가 많으며, 실수로 지우는 것을 방지하기 위해 숨겨져 있다.

### 🔷 섹션4 —  ls -l 명령어로 상세 파일 속성 확인하기

**개념 요약**

- **ls -l:** 파일의 권한, 소유자, 크기, 수정 날짜 등 상세 정보를 리스트 형식으로 출력합니다.
- **출력 정보 의미:**
    - `-rw-r--r--`: 권한 (첫 글자가 `d`면 디렉터리, `-`면 일반 파일)
    - `1`: 하드 링크 수
    - `labex (첫 번째)`: 소유 사용자
    - `labex (두 번째)`: 소유 그룹
    - `0`: 파일 크기 (바이트 단위)
    - `Oct 26 10:30`: 최종 수정 시간

**명령어 / 문법**

```bash
ls -l # total 0 -rw-r--r-- 1 labex labex 0 Oct 26 10:30 sample.txt

ls -la # total 8 drwxr-xr-x 2 labex labex 4096 Oct 26 10:35 . drwxr-xr-x 3 labex labex 4096 Oct 26 10:20 .. -rw-r--r-- 1 labex labex 0 Oct 26 10:30 .hidden_file -rw-r--r-- 1 labex labex 0 Oct 26 10:30 sample.txt
```

**실습 메모**

- `ls -la` 결과에서 `.`과 `..`의 권한이 `d`로 시작하는 것을 통해 디렉터리임을 재확인할 수 있다.

### 🔷 섹션 5 —  상대 경로 (..) 를 사용하여 탐색하고 홈으로 돌아가기

**개념 요약**

- **상대 경로:** 현재 내가 있는 위치를 기준으로 이동합니다.
- **.. (Double Dot):** 현재 디렉터리의 바로 상위(부모) 디렉터리를 의미합니다.
- 인자 없는 `cd`는 항상 `~`로 점프하는 마법의 버튼과 같다.

**명령어 / 문법**

```bash
pwd        # 현재 /home/labex/project 라면
cd ..      # 부모 디렉터리로 이동
pwd        # /home/labex 확인

```

**실습 메모**

- 한 단계 위로 갈 때는 `cd ..`, 두 단계 위는 `cd ../..`를 사용한다.
