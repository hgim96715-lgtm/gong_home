### 🔷 섹션 1 — 파일 생성 및 입력 모드에서 텍스트 작성

**개념 요약**

```
vim 은 모드 기반 편집기
  Normal 모드  → 기본 시작 모드 (명령어 입력)
  Insert 모드  → 텍스트 입력 모드
  Command 모드 → : 로 시작하는 명령어 실행

파일 생성:
  vim 파일명   → 파일 없으면 새로 생성
```

**실습 메모**

```bash
vi test.txt

# Normal → Insert 모드 진입
i    # 커서 앞에서 입력

# Insert → Normal 모드 복귀
ESC
```

---

### 🔷 섹션 2 — :w 및 :wq 를 사용하여 파일 저장 및 종료

**개념 요약**

```text
Normal 모드에서 : 를 누르면 Command 모드 진입
저장 / 종료 명령어는 전부 : 로 시작
```

**실습 메모**

```bash
:w       # 저장 (종료 안 함)
:wq      # 저장 후 종료
:q!      # 저장 없이 강제 종료
```

---
### 🔷 섹션 3 — gg, h, j, k, l 및 / 를 이용한 탐색과 검색

**개념 요약**

```
Normal 모드에서 방향키 대신 hjkl 사용
마우스 없이 키보드만으로 빠르게 이동
```

**실습 메모**

```bash
# 이동
h    # ← 왼쪽
j    # ↓ 아래
k    # ↑ 위
l    # → 오른쪽

gg   # 파일 맨 위로

# 검색
/패턴    # 아래 방향 검색
n        # 다음 검색 결과
```

### 🔷 섹션 4 — dw 로 단어 삭제 및 :q 로 저장 없이 종료

**개념 요약**

```
Normal 모드에서 바로 편집 가능 (Insert 모드 진입 불필요)
d = delete 명령어 / w = word 단위
```

**실습 메모**

```bash
# 삭제
dw       # 단어 삭제 (커서부터 단어 끝까지)

# 저장 없이 종료
:q!      # 강제 종료 (변경사항 버림)
```

### 🔷 섹션 5 — 추가 연습을 위한 대화형 vimtutor 실행

vimtutor = vim 공식 내장 튜토리얼

### 🔷 섹션 6 — nano 편집기 소개

**개념 요약**

```
nano = 모드 없는 편집기
열자마자 바로 타이핑 가능
하단에 단축키 항상 표시 → 외울 필요 없음
초보자 / 빠른 수정에 적합
```

**실습 메모**

```bash
nano welcome.txt     # 파일 열기

# nano 안에서
Ctrl + O    # 저장 (Write Out) 꼭 저장
Ctrl + X    # 종료
Ctrl + W    # 검색
Ctrl + K    # 줄 잘라내기
Ctrl + U    # 붙여넣기
```
