## 문제 분석 & 파이프라인 설계 (Shell logic)

> 각 행에 도시 이름과 해당 주(state)가 쉼표로 구분되어 있는 CSV 파일이 주어집니다.  
> 당신의 작업은 예시와 같이 파일의 **줄바꿈(newline)** 을 **탭(tab)** 으로 바꾸는 것입니다.

1. **Input/Output 데이터 형태:**
    - **Input**: CSV 파일-> 두 값은 **쉼표(,)** 로 구분됨
    - **Output**: 입력에서 사용된 **줄바꿈(newline)** 을 **탭(tab)** 으로 바꿔 출력
2. **핵심 명령어 & 로직 (Pipeline):**

3. **사용할 주요 옵션 (Flags):**
	- `-s` (serial) 여러 줄을 한 줄로 이어붙임 (세로를 가로로 )
	- `-d '\t'` → 구분자를 탭으로 지정

---
## ✅ 정답 스크립트 (Solution)

```bash
#!/bin/bash
paste -s -d '\t'
```

---
## 오답 노트 & 배운 점 (Retrospective)

- **내가 실수한 부분 (Syntax/Spacing):**
	- 처음에 `awk -F` 또는 `OFS`를 사용하려고 함 
	- `OFS`(Output Field Separator)를 바꾸면 될 거라고 생각함
	- 하지만 `OFS`는 **필드 사이 구분자!!!**
-  **새로 알게 된 명령어/옵션:**
    - `OFS` → 필드 사이 (`,` → <code>|</code> 이런 변환할 때 사용)
    - `ORS` → **줄 사이 (핵심!)**

---
## 더 나은 풀이 (One-liner / Efficiency)

```bash
# 숏코딩 / 효율적인 한 줄 명령어
awk 'ORS="\t"'
```
