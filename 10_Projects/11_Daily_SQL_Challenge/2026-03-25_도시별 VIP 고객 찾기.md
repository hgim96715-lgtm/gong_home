## 정답 쿼리 (Solution)

```sql
-- 여기에 작성한 쿼리를 붙여넣으세요.
WITH custom AS (
    SELECT 
        customer_id,
        city_id,
        SUM(total_price - discount_amount) AS cus_total
    FROM transactions
    WHERE is_returned = FALSE
    GROUP BY customer_id, city_id
),

ranked AS (
    SELECT *,
        RANK() OVER (
            PARTITION BY city_id 
            ORDER BY cus_total DESC
        ) AS rnk
    FROM custom
)

SELECT 
    city_id,
    customer_id,
    cus_total AS total_spent
FROM ranked
WHERE rnk = 1;
```
