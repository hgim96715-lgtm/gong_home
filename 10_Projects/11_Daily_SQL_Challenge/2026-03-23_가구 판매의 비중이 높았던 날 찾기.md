## 정답 쿼리 (Solution)

```sql
-- 여기에 작성한 쿼리를 붙여넣으세요.
SELECT 
    order_date,
    COUNT(DISTINCT order_id) FILTER (WHERE category = 'Furniture') AS furniture,
    ROUND(
        COUNT(DISTINCT order_id) FILTER (WHERE category='Furniture') * 1.0 
        / COUNT(DISTINCT order_id) * 100,
        2
    ) AS furniture_pct
FROM records r
GROUP BY order_date
HAVING 
    COUNT(DISTINCT order_id) >= 10
    AND COUNT(DISTINCT order_id) FILTER (WHERE category = 'Furniture') * 1.0 
        / COUNT(DISTINCT order_id) >= 0.4
ORDER BY furniture_pct DESC, order_date;
```
