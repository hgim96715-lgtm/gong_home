##  해결 전략 (Code Before Think)

>  2020년 12월 동안 있었던 모든 주문의 매출 합계가 1000$ 이상인 고객 ID를 출력하는 쿼리를 작성해주세요.
>  쿼리 결과에는 고객 ID가 들어있는 컬럼 하나만 있어야 합니다.

1. **타겟 데이터:** (어떤 테이블에서 무엇을 뽑아야 하는가?)
   - customer_id,order_date-> records
   - 고객의 id -> customer_stats
1. **조건 분석:**
   - 2020년 12월 동안 
   - 모든 주문의 매출 합계
1. **사용할 문법:**
   - WHERE
   - GROUP
---
## 정답 쿼리 (Solution)

```sql
-- 여기에 작성한 쿼리를 붙여넣으세요.
WITH customer_win AS(
SELECT SUM(sales),customer_id
FROM records
WHERE order_date >='2020-12-01' AND order_date<'2021-01-01'
GROUP BY customer_id
HAVING SUM(sales)>=1000
)

SELECT cs.customer_id
FROM customer_stats as cs
JOIN customer_win cw ON cs.customer_id=cw.customer_id
```


## 다른 날 다른 정답쿼리 (복습때 시도)

```sql
SELECT r.customer_id
FROM records r
WHERE EXTRACT(MONTH FROM r.order_date)='12'
GROUP BY r.customer_id
HAVING SUM(r.sales)>=1000
```

## 더 나은 풀이가 있다면?

- 문제에서 요구한게 customer_id 컬럼 하나라서 join 이 필요 없음

```sql
-- 더 나은 풀이가 있을경우 
SELECT customer_id
FROM records
WHERE order_date >= '2020-12-01'
  AND order_date < '2021-01-01'
GROUP BY customer_id
HAVING SUM(sales) >= 1000;
```
