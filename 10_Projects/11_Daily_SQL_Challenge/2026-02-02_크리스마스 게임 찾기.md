##  해결 전략 (Code Before Think)

> `games` 테이블에서 게임 이름에 "Christmas" 또는 "Santa"가 포함된 게임을 찾는 쿼리를 작성해주세요.

1. **타겟 데이터:** (어떤 테이블에서 무엇을 뽑아야 하는가?)
   - games 
   - `game_id` : 게임 ID
   - `name` : 게임 이름
   - `year` : 발매 연도 
1. **조건 분석:**
   - 또는 OR
   - 포함된 
1. **사용할 문법:**
   - WHERE OR
   - ILIKE
---
## 정답 쿼리 (Solution)

```sql
-- 여기에 작성한 쿼리를 붙여넣으세요.
SELECT game_id,name,year
FROM games
WHERE name ILIKE '%Christmas%' OR name ILIKE '%Santa%'
```

---
##  오답 노트 & 배운 점 (Retrospective)

-  **내가 실수한 부분:**
	- `IN`은 **값의 완전 일치 비교**에만 사용하는 연산자인데,
	- 처음에 나는 `ILIKE`처럼 **패턴 매칭(와일드카드 `%`)** 에도 `IN`을 쓸 수 있을 거라고 착각해서 시도해봄
-  **새로 알게 된 함수/꿀팁:**
	- **`ILIKE`는 하나의 패턴만 비교 가능**
	- 여러 패턴을 찾고 싶을 때는 `OR`로 연결
	- 만약 `IN`처럼 묶어서 쓰고 싶다면 ->  **`ILIKE ANY (ARRAY[])`**

> `IN`은 값 비교, `ILIKE`는 패턴 비교 — 둘은 역할이 다르다

## 더 나은 풀이가 있다면?

```sql
-- 더 나은 풀이가 있을경우 
SELECT game_id, name, year
FROM games
WHERE name ILIKE ANY (ARRAY['%Christmas%', '%Santa%']);


-- 정규식 표현으로 구하기
-- `~*` = 대소문자 무시
SELECT game_id, name, year
FROM games
WHERE name ~* '(christmas|santa)';
```
