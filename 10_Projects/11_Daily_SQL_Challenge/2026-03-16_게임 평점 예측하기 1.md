##  해결 전략 (Code Before Think)

> 2015년 이후에 발매한 게임 중 누락된 평점 정보가 있는 게임에 대해서 같은 장르를 가진 전체 게임의 평균 평점과 평균 평론가/사용자 수를 사용해 누락된 정보를 채우는 쿼리를 작성해주세요.
> 쿼리 결과에는 누락된 평점 정보가 있는 게임만 포함되어야 하며, 아래 6개 컬럼이 있어야 합니다. 
> 또한, 평론가와 사용자 평점 평균은 소수점 아래 넷째 자리에서 반올림 해 셋째 자리까지 출력되어야 하고, 사용자 수는 올림하여 자연수로 출력되어야 합니다.

1. **타겟 데이터:** (어떤 테이블에서 무엇을 뽑아야 하는가?)
   - `games` 테이블
	   - `game_id` 게임 ID
	   - ` name` 게임 이름
	   -  `year` 출시 연도

	   - ` critic_count` 평가한 참여한 평론가 수
	   - ` user_score` 사용자 평균 점수 (0-10)
	   - ` user_count` 평가에 참여한 사용자 수

## 정답 쿼리 (Solution)

```sql
-- 여기에 작성한 쿼리를 붙여넣으세요.
WITH avg_group AS (
    SELECT
        genre_id,
        AVG(critic_score) AS avg_crs,
        AVG(critic_count) AS avg_crc,
        AVG(user_score) AS avg_us,
        AVG(user_count) AS avg_uc
    FROM games
    GROUP BY genre_id
)

SELECT
    g.game_id,
    g.name,
    ROUND(COALESCE(g.critic_score, avg_group.avg_crs), 3) AS critic_score,
    CEIL(COALESCE(g.critic_count, avg_group.avg_crc)) AS critic_count,
    ROUND(COALESCE(g.user_score, avg_group.avg_us), 3) AS user_score,
    CEIL(COALESCE(g.user_count, avg_group.avg_uc)) AS user_count
FROM games g
JOIN avg_group
  ON g.genre_id = avg_group.genre_id
WHERE g.year >= 2015
  AND (g.critic_count IS NULL OR g.user_count IS NULL);
```

##  오답 노트 & 배운 점 (Retrospective)

-  **내가 실수한 부분:**
    - 처음에는 **사용자 수 / 평론가 수를 자연수로 출력해야 한다는 조건**을 놓쳐서 `ROUND()`를 사용했다.
