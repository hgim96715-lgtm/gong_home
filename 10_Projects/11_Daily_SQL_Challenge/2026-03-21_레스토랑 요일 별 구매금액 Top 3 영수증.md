## 정답 쿼리 (Solution)

```sql
-- 여기에 작성한 쿼리를 붙여넣으세요.
WITH rank_day AS (
    SELECT 
        day,
        time,
        sex,
        total_bill,
        RANK() OVER (PARTITION BY day ORDER BY total_bill DESC) AS rank
    FROM tips
)

SELECT 
    day,
    time,
    sex,
    total_bill
FROM rank_day
WHERE rank <= 3;
```

## 더 나은 풀이가 있다면?

```sql
-- 더 나은 풀이가 있을경우 
WITH ranked AS (
    SELECT 
        day,
        time,
        sex,
        total_bill,
        ROW_NUMBER() OVER (PARTITION BY day ORDER BY total_bill DESC) AS rn
    FROM tips
)

SELECT 
    day,
    time,
    sex,
    total_bill
FROM ranked
WHERE rn <= 3;
```

> **정확히 N명만** 뽑는 거니깐 ROW_NUMBER가 더 맞지 않을까?
