##  해결 전략 (Code Before Think)

> 두 사용자 ID의 합이 상위 0.1%에 들어오는 모든 친구 관계를 출력하는 쿼리를 작성해주세요.

1. **타겟 데이터:** (어떤 테이블에서 무엇을 뽑아야 하는가?)
   - `edges` 테이블
	   - `user_a_id` 친구 관계인 사용자 ID (A)
	   - ` user_b_id`  친구 관계인 사용자 ID (B)
1. **조건 분석:**
   - ID 합 기준으로 상위 0.1%에 해당하는 관계 추출
1. **사용할 문법:**
   - `user_a_id + user_b_id` 
   - `PERCENT_RANK()`
---
## 정답 쿼리 (Solution)

```sql
-- 여기에 작성한 쿼리를 붙여넣으세요.
WITH pct AS(
    SELECT 
        user_a_id,
        user_b_id,
        user_a_id + user_b_id AS id_sum,
        PERCENT_RANK() OVER (ORDER BY user_a_id + user_b_id ASC) AS percent_rank
    FROM edges
)

SELECT 
    user_a_id,
    user_b_id,
    id_sum
FROM pct
WHERE percent_rank <= 0.001
```
